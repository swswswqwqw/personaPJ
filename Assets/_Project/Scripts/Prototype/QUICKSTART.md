# 残響都市アマネ — プロトタイプ クイックスタート

## 起動方法

1. Unity Hub で `personaPJ` フォルダを開く（Unity 6000.3.2f1）
2. メニューバー → **Amane** → **プロトタイプシーンを作成**
3. **Play ボタン (▶)** を押す

## 操作

### タイトル画面
- 「未読を開く」→ ゲーム開始（フィールドへ）
- 「日記を読む」→ セーブデータがあればロード

### フィールド画面
- 右上: カレンダー（日付・曜日・天候・時間帯・AP）
- 中央: 行動選択パネル
  - **勉強する** → 知性+5（AP1消費）
  - **誰かと過ごす** → 灯里との会話イベント（AP1消費）
  - **バイトする** → ことのは+3, 静けさ+2（AP1消費）
  - **瞑想する** → 静けさ+5（AP1消費）
  - **未言界へ潜行する** → 戦闘へ（AP全消費）
  - **帰宅する** → 次の時間帯へ

### 会話
- **Space** or **Enter** → 次の台詞へ
- 選択肢 → マウスクリック（ボンドポイント付与）

### 戦闘
- **攻撃** → 敵を選んで通常攻撃
- **スキル** → スキル一覧 → 対象選択
- **防御** → ターンを防御で消費
- **言継ぎ** → 仲間を選んで手番を渡す（ダメージ+50%/+25%刻み）
- **総告白** → 全敵DOWN時のみ発動（全体大ダメージ）
- **逃走** → フィールドへ戻る

## テスト実行
メニューバー → **Amane** → **全ロジックテスト実行**
