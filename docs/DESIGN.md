## 1. タイトル・コアコンセプト

**ゲームタイトル（日本語）**: 残響のアストラル
**ゲームタイトル（英語）**: Astral Echoes
**キャッチコピー**: 「沈めた声が、世界を壊す」

**コアテーマ**:
抑圧された感情は消えない——それは「残響」となって異世界に結晶化し、やがて現実を侵食する。
人は自分の声と向き合わなければ、本当の繋がりを得られない。

**プレイヤーに最終的に感じさせたい感情**:
「弱さを見せることは強さだ」という実感。ゲームを終えたとき、プレイヤーが自分の大切な人に正直に話したくなる——そんな余韻を残す。

## 2. 技術スタック

**エンジン**: Unity 6（6000.x LTS）
**言語**: C#

- C#ネイティブで直接実装可能
- 2D/2.5D JRPG開発のエコシステムが成熟
- ScriptableObjectによるデータ駆動設計が強力
- UniTaskによるasync/await対応でコルーチン不要
- Addressablesによる効率的なアセット管理

- UniTask（async/await）

## 3. 世界観・舞台設定

### 舞台

**都市**: 響ヶ崎市（ひびきがさき-し）
神奈川県の架空の海沿いの街。鎌倉・江ノ島エリアをベースにしている。古い寺社と現代アートギャラリーが混在する文化都市。海風が独特の音響現象を起こすことで知られ、「響きの街」と呼ばれる。

**学校**: 星嶺学園（せいれいがくえん）
進学校。学力ランキングへの執着が強く、生徒の精神的プレッシャーが社会問題になりかけている。海が見える高台に位置し、放課後の屋上からの景色が象徴的なロケーション。

### 二重世界

**現実世界**: 日本の学校生活。授業、部活、バイト、人間関係。普通の日常の中に、人々の抑圧された感情が見え隠れする。

**異世界「アストラル」**:
人間の抑圧された感情が結晶化して形成される並行次元。ユング心理学の集合的無意識が物質化した世界。感情の周波数が物理的な形を持ち、音が結晶構造となり、光の川が流れ、幾何学模様が浮遊する。

各個人の強い抑圧は「共鳴宮殿（レゾナンス・パレス）」というダンジョンを形成する。宮殿の構造と美術は、その人物の内面世界を反映する。

### アストラルが存在する理由

アストラルは人類の意識の感情的な基盤として太古から存在してきた。響ヶ崎市は「共鳴点」——現実とアストラルの境界が薄い場所——の上に位置している。近年のSNS社会、デジタル孤立、感情抑圧の増加がアストラルを不安定にし、「不協和（ディソナンス）」と呼ばれる負の感情から生まれた怪物が現実に漏れ出し始めている。

### アストラルの視覚的イメージ

- **基調**: 深い群青と紫を基本色とした、音が物質化した世界
- **感情別アクセント色**:
  - 怒り = 深紅
  - 悲しみ = 淡い青
  - 恐怖 = 病的な黄緑
  - 喜び = 琥珀色の光
  - 絶望 = 灰色の霧
- **質感**: 結晶、音波のリング、脈動する光脈
- **空気感**: 水中にいるような静寂と、遠くから聞こえる微かな共鳴音
- 各宮殿は固有のビジュアルテーマを持つ（例：怒りを抑圧する人の宮殿は茨の檻、悲しみに溺れる人の宮殿は凍った海洋）

### アストラルへの侵入条件

主人公は「共鳴アプリ（レゾナンス・アプリ）」を使用する。臨死体験後に突然スマホに出現した謎のアプリで、開発者不明、アップデートなし、削除不可能。対象者の感情周波数に集中することで、その人物の共鳴宮殿へのゲートを開く。対象者の近くにいるか、強い感情的つながりが必要。

### 主人公

**名前**: 朝霧 蓮司（あさぎり れんじ）
**年齢**: 17歳、高校2年生
**外見**: 乱れた黒髪に一筋の銀（臨死体験の痕跡）。鋭いが優しい目。常にヘッドフォンを首にかけている。

**特殊能力「調律転換（ハーモニック・シフト）」**:
異なる感情周波数に同調し、それを「残響体（エコー）」という戦闘能力として顕現させることができる。他のパーティメンバーが1体の固有エコーしか持てないのに対し、蓮司は複数のエコーを切り替えられる。臨死体験が彼の感情の壁を粉砕し、全ての周波数と共鳴できるようになったため。

**背景**: 前の学校でいじめられていた同級生を助けようとしたが、事態を悪化させたと責められた。転校中の事故で臨死体験を経て、人々の抑圧された感情が「聞こえる」能力を得た。叔母の朝霧雫が営むカフェ「残響堂」に身を寄せ、星嶺学園に転入する。

## 4. ゲームシステム設計

### 時間システム

**1日の構造**:

| 時間帯 | 行動ポイント | 可能な行動 |
|--------|------------|-----------|
| 朝（Morning） | 0（ナラティブ専用） | 朝礼イベント、通学時会話 |
| 授業（Class） | 0（固定イベント） | 授業応答で洞察UP、ポップクイズ |
| 放課後（After School） | 1 | アチューンメント、部活、街探索、バイト、アストラル潜入 |
| 夜（Evening） | 1 | 勉強、限定アチューンメント、アストラル探索、夜限定スポット |
| 深夜（Late Night） | 0.5（限定行動） | 勉強、アイテム制作、決意が高ければ外出イベント |

**時間経過イベントトリガー**（6つ）:

1. **期末試験（Midterms/Finals）**: 約6週間毎。洞察チェック、学年ランキング発表。高順位で全アチューンメントにボーナス。
2. **文化祭（Cultural Festival）**: 10月。特別アチューンメントイベント多数、ユニークな選択肢。
3. **満月の共鳴（Full Moon Resonance）**: 毎月。アストラルが最も危険になるが、レア素材と限定イベントが出現。現在の宮殿クリア期限。
4. **海鳴りの日（Day of the Sea Roar）**: 季節毎（6月、9月、12月、3月）。世界の境界が最も薄くなり、特別なストーリーイベント発生。
5. **季節イベント**: 七夕、夏祭り、ハロウィン、クリスマス、正月、バレンタイン。アチューンメント深化の好機。
6. **不協和溢出（Dissonance Overflow）**: 宮殿を期限内にクリアしないと発生。ディソナンスが現実に漏れ出し、バッドイベント連鎖→バッドエンド分岐。

**月次デッドライン**:
各ストーリーアークには攻略対象がおり、その共鳴宮殿を次の満月までに攻略する必要がある（約30日間）。満月はディソナンスを危険なレベルまで増幅する。期限失敗で対象者が精神崩壊し、現実世界に連鎖的悪影響（バッドエンド分岐）。

### 戦闘システム

**基本形式**: ターン制コマンドバトル（ペルソナ5の「One More」システムを参考）

**属性と弱点システム**:

| 属性 | 効果 | 状態異常 |
|------|------|---------|
| 斬撃（Slash） | 物理・刃系 | — |
| 打撃（Strike） | 物理・打系 | — |
| 貫通（Pierce） | 物理・突系 | — |
| 火炎（Fire） | 魔法・火 | 炎上（DOT） |
| 氷結（Ice） | 魔法・氷 | 凍結（行動不能） |
| 雷電（Thunder） | 魔法・雷 | 感電（テクニカルコンボ起点） |
| 疾風（Wind） | 魔法・風 | ダウン促進 |
| 光明（Light） | 魔法・光 | 即死（低確率）、回復 |
| 暗黒（Dark） | 魔法・闇 | 即死（低確率）、デバフ |
| 共鳴（Resonance） | 感情ダメージ | 物理防御無視。ディソナンスに特効 |
| 虚無（Null/Almighty） | 全貫通 | 弱点・耐性なし |

**耐性レベル**: 弱点 / 通常 / 耐性 / 無効 / 吸収 / 反射

**弱点ヒット時の効果「One More」**:
弱点を突くと敵がダウンし、攻撃者に追加ターンが与えられる。クリティカルヒットでも同様。全ての敵がダウンすると「全共鳴撃」が発動可能に。

**全共鳴撃（Full Resonance Attack）**:
ペルソナのAll-out Attack相当。全敵ダウン時に発動可能。
- 演出: 画面がガラスのように砕け散る
- パーティメンバーが協調攻撃、固有キャラアートのスプラッシュ表示
- フィニッシュ画面: アクティブキャラのスタイリッシュポーズ＋個人的な一言
- ダメージ: パーティの感情共鳴レベル（戦闘中に蓄積）に基づく大ダメージ

**主人公の特殊能力「残響転換（エコー・シフト）」**:
蓮司は複数の残響体（エコー）を切り替えて戦闘できる（ペルソナ切り替え相当）。
各エコーが持つもの:
- 感情周波数（属性親和性）
- 固有スキル
- ステータス補正
- エコーは「共鳴室（レゾナンス・チェンバー）」で合体し、より強力なエコーを生成可能

## 6. ストーリー構成

### 3幕構成

**第1幕: 覚醒（4月〜6月）**「無視できない声」
- 蓮司が星嶺学園に転入
- 共鳴アプリの発見、エコー能力の覚醒
- 初のアストラル潜入：いじめられている生徒の崩壊する宮殿を救出
- 凛花と烈がエコーに覚醒
- 最初の本格攻略対象：成績を操作している教師・犬神（6月末期限）
- テーマ：「聞こえてしまった声は、もう無視できない」

**第2幕: 不協和（7月〜11月）**「音符と音符の間の沈黙」
- パーティ拡大（詩織加入、アチューンメント対象増加）
- 複数の宮殿をエスカレートする緊張感で攻略:
  - 8月: 児童労働を搾取する企業幹部（遊馬の父の関連会社）
  - 9月: 感情操作を拡散する人気インフルエンサー（サクヤの世界と接続）
  - 10月: 学校の悲劇を隠蔽する政治家（マコトの過去と接続）
  - 11月: 絶望を餌にするカルト教団の主（ナギを脅かす）
- 宮殿攻略で解放された感情エネルギーを「何か」が収集していることが判明
- **中盤の転換点**: 蓮司自身にも宮殿があることが発覚——抑圧した罪悪感が肥大化している
- テーマ：「音符の間にある沈黙こそが、音楽を形作る」

**第3幕: 共鳴（12月〜3月）**「お前にしか歌えない歌」
- 真の敵の正体判明：「沈黙（サイレンス）」——人類の集合的な感情抑圧から生まれた根源的存在
- 沈黙は、攻略済み宮殿から解放された感情エネルギーを吸収して成長していた
- 蓮司は自身の宮殿に立ち向かう必要がある（1月期限）——最難関ダンジョン
- 最終決戦は全アチューンメントの絆の強さに依存
- 結末はゲーム全体を通じたプレイヤーの選択に左右される
- テーマ：「全ての声には意味がある。お前の声にも」

## 付録: 参考資料

### ペルソナシリーズの設計から学んだこと

- **UIデザイン**: P5のコンセプトは「ポップパンク」。赤をメインカラーとして徹底し、他の色を極力排除。メニュー画面の白いラインで視線誘導。階層認識のためにレイアウトとアングルを変化。流動的な明度変化で重要情報を強調。
- **コープシステム**: ランクアップでコープアビリティ習得。アルカナ最上位ペルソナの解禁。対応アルカナ所持で好感度ボーナス。ペルソナ作成時の経験値ボーナスに直結。
- **戦闘**: One Moreシステム——弱点で追加ターン、全ダウンでAll-Out Attack。少数の属性で明確な弱点/耐性の相性。パーティメンバーは各1属性専門、主人公のみ切り替え可能。
- **時間**: 有限の行動ポイントで「選択と犠牲」を生む。月次デッドラインで常に戦略的プレッシャー。
- **人間パラメータ**: 知識・度胸・器用さ・優しさ・魅力の5項目。日常行動で上昇。コープ・イベントの解放条件として機能。
