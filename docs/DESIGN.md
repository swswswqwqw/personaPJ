## 1. タイトル・コアコンセプト

**ゲームタイトル（日本語）:** 空蝉のアルカナ
**ゲームタイトル（英語）:** ARCANA OF HOLLOWS

**キャッチコピー:** 「その仮面の裏に、本当のお前がいる」

**コアテーマ:**
人は何重もの仮面を被って生きている。最も深い仮面を剥がしたとき、そこにあるのは空洞か、それとも本当の自分か——その問いに向き合う物語。

**プレイヤーに最終的に感じさせたい感情:**
「自分の弱さを認めることは、強さの始まりだった」という静かな確信。
ゲームを終えた後、プレイヤーが自分自身の"仮面"について考え、それでも人と繋がろうとする勇気を感じること。

## 2. 技術スタック

**エンジン:** Unity 2022.3 LTS（Universal Render Pipeline）

**選定理由:**
- C#でのロジック記述がペルソナ的な複雑なステートマシン管理に最適
- UniTask + DOTweenによるアニメーション・非同期処理が演出密度の高いJRPGに適合
- ScriptableObjectによるデータ駆動設計がキャラクター・スキル・イベントの大量データ管理に有効
- 2D UIの表現力が高く、ペルソナ5的なスタイリッシュUIの実現に適している
- TextMeshProによるリッチテキスト表現（台詞の感情表現）

**必須パッケージ:**
- UniTask（async/await による非同期処理）
- DOTween Pro（UIアニメーション・カットイン演出）
- TextMeshPro（テキスト表示・演出）
- Addressables（アセット管理・シーン遷移）
- Cinemachine（カメラワーク）
- Input System（入力管理）
- Unity UI Toolkit（メニュー・HUD）

## 3. 世界観・舞台設定

### 舞台
**都市:** 潮鳴市（しおなりし）
- 神奈川県の架空の海沿いの街（鎌倉・江ノ島エリアがモデル）
- 古い寺社と新しい商業施設が共存する、どこか懐かしくて寂しい街
- 海風が常に吹いており、「潮の音が聞こえる街」として知られる
- 夕暮れの海岸線が象徴的なビジュアル

**学校:** 私立 曙光学院（しょこうがくいん）
- 海を見下ろす丘の上に建つ中高一貫校
- 自由な校風で知られるが、内部では見えない序列と同調圧力がある
- 校舎は古い洋館風で、渡り廊下からは海が見える

### 二重世界

**異世界名:** 空蝉界（うつせみかい / The Hollow）

**設定:**
空蝉界は現実世界の「裏側」に広がる並行世界。人々が日常の中で抑圧した感情——言えなかった本音、見せなかった怒り、隠した悲しみ——が沈殿し、結晶化した世界。

**視覚的イメージ:**
- 現実世界を鏡写しにしたような構造だが、全てが半透明の硝子と結晶でできている
- 基調色は深い藍色と琥珀色。感情の「核」だけが鮮やかに発光する
- 空は常に黄昏時のような薄暮。水面のような揺らぎが空間全体にある
- 音は反響し、遠くで誰かの声にならない叫びがこだまする

**異世界が存在する理由:**
人間の感情は消滅しない。抑圧された感情は現実から「剥離」し、空蝉界に沈殿する。通常これは自然なプロセスだが、近年、人々の感情抑圧が限界を超え、空蝉界と現実の境界が薄くなっている。このまま放置すれば、空蝉界が現実を侵食し始める（＝「蝕み」現象）。

**侵入条件・方法:**
- 主人公は「共鳴者」——他者の抑圧感情と共鳴し、空蝉界への「裂け目」を感知できる特異体質
- 特定の人物の感情が極限まで抑圧されると、その人物の周囲に「空蝉の門」が出現する
- 門はスマートフォンの専用アプリ「カレイドスコープ」で可視化・開門できる（ある日突然インストールされる）
- 仲間も主人公との「共鳴」を通じて空蝉界に入れるようになる

### 主人公

**名前:** 朝霧 蓮司（あさぎり れんじ）※デフォルト名・変更可
**年齢:** 17歳（高校2年生）
**外見:** やや長めの黒髪、鋭いが寂しげな目。制服のネクタイをいつも少し緩めている。

**特殊能力:** 共鳴（レゾナンス）
- 他者の「本当の声」を感じ取る力。言葉の裏にある感情が、音として聞こえる
- この力のせいで前の学校で「人の心を読む気持ち悪いやつ」と孤立し、転校してきた
- 空蝉界では、この力が「反響体（エコー）」の召喚として発現する
- 複数の反響体を切り替えて使役できる唯一の存在

**物語上の秘密:**
蓮司の共鳴能力は先天的なものではない。幼少期に母親が空蝉界に飲み込まれる瞬間を目撃し、その時に境界の力の一部が流入した。母親は今も空蝉界のどこかにいる——これが物語後半の核心的動機となる。

### 4-2. 戦闘システム

**基本形式:** ターン制コマンドバトル（プレスターン変種）

**属性システム（8属性）:**

| 属性 | 記号 | 色 | 対応感情 |
|------|------|-----|---------|
| 火炎 | 🔥 | 赤橙 | 怒り |
| 氷結 | ❄️ | 青白 | 悲しみ |
| 疾風 | 🌀 | 緑 | 不安 |
| 雷電 | ⚡ | 黄 | 焦り |
| 念動 | 🔮 | 紫 | 執着 |
| 核熱 | ☢️ | 桃 | 恥 |
| 物理 | ⚔️ | 灰 | — |
| 万能 | ✦ | 白金 | — |

**弱点ヒット時の効果:**
- 1MORE（追加行動権を獲得、ペルソナと同一）
- 共鳴ゲージが1段階上昇
- 弱点を突かれた敵はダウン状態（1ターン行動不能）

**共鳴ゲージ → ハーモニーブレイク:**
全敵がダウン状態になると「ハーモニーブレイク」（All-out Attack相当）が発動可能。
- 画面が一瞬モノクロになり、パーティ全員のカットインが入る
- 各キャラの攻撃モーションが連続で流れ、最後に主人公の決めポーズ
- 演出は心弦ランクが高いキャラほど派手になる（システムとナラティブの融合）
- 発動時にスタイリッシュなスプラッシュアート表示（ペルソナ5のAll-out Attackオマージュ）

**反響体（エコー）:** ペルソナに相当する召喚存在
- 主人公のみ複数の反響体を所持・切り替え可能
- 仲間は各自1体の固有反響体を持つ（覚醒・超覚醒で進化）
- 反響体はギリシャ神話・世界の神話をモチーフとする
- 合体（共鳴融合）で新たな反響体を生成可能

**戦闘の感情設計:**
- 弱点を突く → 「やった！」の快感（即座のフィードバック）
- 1MORE連鎖 → テンポの良い爽快感
- ハーモニーブレイク → カタルシスの頂点
- 被弱点 → 緊張感と戦略の見直し
- ボス戦 → 絶望 → 突破口発見 → 逆転の感動

### 4-3. 心弦（しんせん）システム — 社会リンク相当

**システム名:** 心弦（Heart Strings）
**コンセプト:** 他者と「同じ周波数」に合わせる（チューニング）ことで絆を深める

**最大ランク:** 10

**心弦の仕組み:**
- 放課後・夜の自由行動で特定キャラと過ごすとランクUP可能
- 会話選択肢で「共鳴」（正解）を選ぶとポイント加算
- ランクアップ時に専用イベントシーン（フルボイス想定で設計）
- 特定ランクで戦闘報酬が解放される
- ランク10到達で最終イベント＋究極報酬

## 6. ストーリー構成

### 3幕構成

**第1幕：覚醒（4月〜8月）**
蓮司が潮鳴市に転校し、空蝉界の存在を知り、仲間を集める。各キャラの空蝉（ダンジョン）を攻略しながら、「なぜこの街で空蝉界が活性化しているのか」の謎が提示される。

**第2幕：真実（9月〜1月）**
空蝉界の活性化は偶然ではなく、何者かが意図的に市民の感情を抑圧し、空蝉界のエネルギーを収集していたことが判明。黒幕は曙光学院の理事長・神代 総一郎。彼は「感情のない世界こそ平和な世界」という歪んだ理想を持ち、空蝉界と現実の融合を企てている。
同時に、蓮司は母の消失の真相に近づく。母は神代の最初の実験の被害者だった。

**第3幕：決断（2月〜3月）**
神代の計画が最終段階に入り、潮鳴市全体が空蝉界に飲み込まれかける。蓮司は母を救出するために最深部へ向かうが、そこで「母は既に空蝉界と融合し、境界そのものになっている」という事実に直面する。母を救えば空蝉界が崩壊し、全ての空蝉化した人々が消える。母を諦めれば世界は救える。

### エンディング分岐

**バッドエンド：「空蝉の王」**
- 条件：月次ボス2体以上を期限切れにする、または最終選択で「母を救う」を選びつつ心弦が不十分
- 内容：蓮司が空蝉界の力に飲み込まれ、新たな空蝉界の核となる。仲間たちは現実に戻されるが、蓮司は二度と戻れない。最後の映像は、蓮司の空蝉が静かに微笑むシーン
- 感情：喪失感と後悔

**ノーマルエンド：「潮騒の記憶」**
- 条件：全月次ボスを撃破し、最終選択で「世界を救う（母を諦める）」を選ぶ
- 内容：空蝉界は消滅し、潮鳴市は救われる。しかし仲間たちの空蝉界での記憶は薄れていき、蓮司は再び「普通の転校生」に戻る。春の卒業式で、かすかに残る絆の温もりを感じながら一人で歩き出す
- 感情：切なさと静かな希望

**トゥルーエンド：「空蝉を超えて」**
- 条件：全月次ボス撃破＋パーティ全員の心弦ランク10＋特定の隠しイベントクリア
- 内容：仲間全員の心弦の力が共鳴し、「第三の選択肢」が出現。母を空蝉界から分離しつつ、空蝉界を破壊するのではなく「人間の感情の一部として再統合」する。蓮司は共鳴能力を失うが、もう力がなくても人と繋がれることを知っている。1年後の潮鳴市で、全員が変わらず友人として再会するエピローグ
- 感情：温かい涙と「繋がることの意味」の確信

### どんでん返し

**どんでん返し1：「凛の正体」（11月）**
- **伏線:** 凛は夜しか学校に現れない。空蝉界の話に異常に詳しい。蓮司の共鳴能力に反応しない
- **回収:** 凛は人間ではなく、蓮司の母が空蝉界から送り出した「感情の断片」。母の愛情が人の形を取った存在。凛が消えることは母の感情の一部が消えることを意味する
- **感情的インパクト:** 仲間だと思っていた存在が「母の祈り」だったという衝撃と感動

**どんでん返し2：「神代の真意」（1月）**
- **伏線:** 神代は冷徹だが、理由なく人を傷つけることはしない。研究室に一枚の写真がある（背を向けた人物）
- **回収:** 神代もかつて共鳴者だった。最愛の人の感情を受け取りすぎて壊れた過去を持つ。「感情のない世界」は、二度と愛する人の痛みで壊れないための防御だった。神代は敵であり、蓮司の未来の姿でもある
- **感情的インパクト:** 「悪」が理解可能な痛みから生まれていたことへの複雑な共感

**どんでん返し3（トゥルーエンド限定）：「共鳴の意味」（3月）**
- **伏線:** 蓮司の共鳴能力は「一方的に他者の感情を受け取る」ものとされてきた
- **回収:** 最終戦で判明——共鳴は双方向だった。蓮司が仲間の感情を受け取っていた時、仲間もまた蓮司の感情を受け取っていた。蓮司が孤独に耐えてきた全ての瞬間、仲間は無意識にそれを感じ、だからこそ蓮司のそばにいることを選び続けた
- **感情的インパクト:** 「一人で背負っていたと思っていたものは、最初から分かち合っていた」

**ペルソナに勝っている点:**

**このゲームが最も力を入れるべき点:**
演出密度。設計がどれだけ良くても、プレイヤーに「伝わらなければ」意味がない。一つ一つの演出に感情的意図を込めること。

## 設計メモ（意思決定の記録）

### 2026-06-10 初期設計完了
- **エンジン選定:** Unity 2022.3 LTS。C#でのロジック記述とUIの表現力を重視。Unreal Engineは3Dリッチな表現には強いが、ペルソナ的な2D UIの演出密度を優先してUnityを選択
- **世界観:** 「空蝉」（うつせみ＝蝉の抜け殻）をメタファーに採用。ペルソナの「仮面」と同根のテーマだが、「中身が空っぽかもしれない」という不安を追加
- **主人公の能力:** 「共鳴」は受動的な力（他者の感情を感じ取る）として設計。ペルソナ主人公の万能感を抑え、「他者の痛みを背負う」コストを強調
- **捨てた案:** 異世界をSNS空間にする案（現代的だが陳腐化リスク）、主人公をAI的存在にする案（感情移入が難しい）
