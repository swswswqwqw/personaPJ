## 1. タイトル・コアコンセプト

**ゲームタイトル**: 残響のアルカディア（Arcadia of Echoes）

**キャッチコピー**: 「言えなかった言葉が、世界を蝕む。」

**コアテーマ**: 人と人の間に残る「言えなかった言葉」の重さと、それを乗り越えて本当の声を届ける勇気。

**プレイヤーに最終的に感じさせたい感情**: 自分の人生にも「言えていない言葉」があることに気づき、それを伝えたいと思う衝動。ゲームを閉じた後、誰かに連絡を取りたくなる作品。

**面白さの公式における位置づけ**:
- 感情移入 → 「言えなかった言葉」は誰もが持つ普遍的経験
- システム融合 → 共鳴（社会リンク）が戦闘力と直結
- 時間の希少性 → 学園生活の1年間、全員とは深まれない
- 演出密度 → 音波・残響をモチーフにした独自UI

## 2. 技術スタック

**エンジン**: Unity 2022.3 LTS（Long Term Support）
**言語**: C#

**選定理由**:
- 2D演出とUI特化の制御が柔軟（ペルソナ風UIの再現に最適）

## 3. 世界観・舞台設定

### 舞台

**都市名**: 澄波市（すみなみし）
- モデル: 神戸と横浜を混ぜた架空の港町
- 丘陵地に広がる住宅街、海沿いの商店街、古い灯台
- 「音が綺麗に響く街」として知られる（音響的な地形）

**学校名**: 澄波市立 聖碧学園（せいへきがくえん）
- 丘の上に建つ、海が見える学校
- 吹奏楽部が全国レベルで有名（音楽のモチーフ）
- 旧校舎に「放送室の幽霊」の噂がある（伏線）

### 二重世界

**異世界名**: 残響界（ざんきょうかい / Echo Realm）

**設定**: 人間が「本当は伝えたかったのに言えなかった言葉」は消えずに残響として蓄積される。残響が一定量を超えると現実世界を侵食し、「残響界」という歪んだ鏡像世界を生み出す。

**存在理由**: 澄波市の地下に眠る古代の「共鳴石」が、人々の感情を増幅している。数百年前から言葉にならなかった感情を吸収し続け、臨界点に達しつつある。

**視覚的イメージ**:
- 現実世界の風景が音波で歪んだような世界
- 色彩: 深い藍色の空に、オーロラのような感情の光が揺らめく
- 建物や道が音叉のように振動し、不安定に浮遊している
- 空気中に「聞こえない声」がノイズとして漂う（視覚的にはパーティクル）
- BGMは常に残響（リバーブ）がかかった不安定な音響

**侵入条件**: 「黄昏刻（たそがれどき）」— 日没の瞬間、約3分間だけ境界が薄くなる。主人公が「共鳴の力」を使うことで、パーティメンバーと共に残響界へ突入できる。

### 主人公

**名前**: 蒼葉 奏（あおば かなで）※プレイヤー変更可能
**年齢**: 17歳（高校2年生）
**外見**: 左耳に小さなイヤーカフ（実は共鳴石の欠片）

**特殊能力**: 「残響聴覚」— 他人が言えなかった言葉が「音」として聞こえる。これにより複数の「共鳴体（きょうめいたい）」を召喚・切り替えできる（ペルソナの複数所持に相当）。

**能力の理由**: 1年前に行方不明になった姉・蒼葉 律（あおば りつ）が残した「イヤーカフ」を着けてから、残響が聞こえるようになった。イヤーカフの正体は共鳴石の欠片。

**背景**: 姉の失踪後、母親との関係が冷え込んでいる。母は奏を「律の代わり」として見ており、奏は「自分自身」として見てほしいが言えない。これが主人公自身の「言えなかった言葉」。

## 4. ゲームシステム設計

### 4.1 時間システム

**ゲーム期間**: 4月〜翌3月（1学年間）

**1日の構造**:
| 時間帯 | 行動ポイント | 主な活動 |
|---|---|---|
| 朝 | 0（イベントのみ） | 登校イベント・朝の会話 |
| 授業 | 0（選択肢のみ） | 授業中の質問応答→知恵UP |
| 放課後 | 1 | 共鳴（社会リンク）・部活・勉強・買い物 |
| 夜 | 1 | 残響界探索・自室で能力強化・アルバイト |
| 深夜 | 0（条件付き） | 度胸が高いと行動可能。特殊イベント |

**時間経過イベントトリガー**:
1. 5月第2週: 中間テスト（知恵が一定以上で順位UP→共鳴解放）
2. 7月20日: 夏祭り（特定キャラとの共鳴イベント発生条件）
3. 8月末: 夏休み終了前の「最後の夜」（パーティ全員イベント）
4. 10月: 学園祭（選択によって複数の共鳴ルートが分岐）
5. 12月24日: クリスマスイベント（恋愛ルート分岐の決定点）
6. 1月: 共鳴石の暴走開始（ストーリー加速・自由時間減少）
7. 3月: 卒業式前日（最終選択）

**月次デッドライン**: 各月に「共鳴崩壊」の期限があり、それまでに該当する残響界のダンジョンをクリアしなければゲームオーバー。期限は約2〜3週間の猶予がある（ペルソナ4のテレビ救出と同構造）。

### 4.2 戦闘システム

**基本形式**: ターン制コマンドバトル（素早さ順行動）

**属性システム**（8属性）:
| 属性 | 対応する「言えなかった感情」 | 色 |
|---|---|---|
| 炎（Blaze） | 怒り — 伝えられなかった憤り | 赤橙 |
| 氷（Frost） | 悲嘆 — 凍りついた悲しみ | 青白 |
| 雷（Volt） | 衝動 — 抑えつけた本能 | 黄金 |
| 風（Gale） | 自由 — 口にできなかった本音 | 翠緑 |
| 光（Lux） | 希望 — 言えなかった励まし | 純白 |
| 闇（Nox） | 秘密 — 隠し通した真実 | 深紫 |
| 念（Psyche） | 執着 — 手放せなかった想い | 桃色 |
| 物理（Strike） | — | — |

**弱点ヒット「残響撃（エコーストライク）」**:
- 敵の弱点属性で攻撃→「1-More」（追加ターン獲得）
- ダウンした敵に追撃可能
- 全敵ダウン→ 「共鳴一斉（ハーモニーブレイク）」発動可能

**共鳴一斉（ハーモニーブレイク）**:
- 全体攻撃の演出名。パーティ全員が共鳴体と同調し、一斉攻撃を放つ
- 演出: 画面が音波で割れ、パーティメンバーのカットインが音楽のビートに合わせて次々表示
- 共鳴ランクが高いメンバーほどダメージ倍率UP（システムとナラティブの融合点）

**主人公の特殊能力「共鳴体スイッチ」**:
- 戦闘中に共鳴体を切り替え可能（ペルソナチェンジ相当）
- 共鳴体は「残響界で倒した敵の感情を昇華」して入手（ペルソナ説得→共鳴による昇華）
- 共鳴体合体（フュージョン）: 「調律（チューニング）」と呼ぶ。調律室で2体以上の共鳴体を融合

**戦闘の感情設計**:
- 弱点ヒット時: 画面が一瞬スローになり、ヒット音が「快感」を生むよう設計
- 共鳴一斉: プレイヤーが「やった！」と声を出すレベルの爽快感
- 全滅時: 静かに画面が暗転。「言葉は、届かなかった。」のテキスト表示

### 4.3 共鳴（社会リンク）システム

**システム名**: 共鳴（きょうめい / Resonance）

**最大ランク**: 10

**核心設計**: 共鳴ランクを上げる＝そのキャラクターの「言えなかった言葉」を一緒に解きほぐすこと。プレイヤーの選択肢は「正解」を選ぶゲームではなく、「この人にどう向き合うか」を選ぶ体験。

**重要ルール**:
- 「慰める」が常に正解ではない。時に「厳しい言葉」が共鳴を深める
- ランク5で「壁」がある。キャラクターが本当の傷を見せる瞬間。ここでの選択で後半の展開が変化
- ランク10で「言えなかった言葉」がついに言葉になる。このシーンが各キャラクターの最大のカタルシス

**I. 桜庭 凛（さくらば りん）— 女帝**

- 表の顔: 完璧な生徒会長。成績トップ、容姿端麗、誰にでも優しい。学園の太陽。
- 隠された傷: 母親が末期がんで闘病中。誰にも言えず、一人で全てを背負っている。「弱い自分」を見せることが何より怖い。
- 言えなかった言葉: 「助けて」
- 成長アーク:
  1. 完璧な仮面（ランク1-2）: 隙のない優等生として主人公と接する
  2. 小さな綻び（ランク3-4）: 授業中に一瞬だけ見せる疲労の影
  3. 壁（ランク5）: 母の見舞いの場面を主人公に目撃される。激しく拒絶
  4. 受容（ランク6-8）: 少しずつ「完璧でなくていい」と理解し始める
  5. 解放（ランク9-10）: 友人の前で初めて泣く。「助けて」と言える
- 代表台詞:
  - 怒り: 「…見たでしょ。見たんでしょ？ 忘れて。今すぐ忘れてよ。」
  - 傷: 「私が泣いたら、誰がお母さんの前で笑うの。」
  - 成長: 「完璧じゃなくても、ここにいていいんだね。」
  - 絆: 「あなたがいたから、私は初めて……助けてって言えた。」
- 戦闘スタイル: 氷属性 + 回復。冷静な分析で敵の弱点を見抜くサポーター
- コープ報酬: 共鳴体に回復スキルボーナス / 凛の固有スキル「氷華の守り」解放

**II. 鷹司 蓮（たかつかさ れん）— 塔**

- 表の顔: 不良。授業はサボり、屋上で寝てばかり。何にも興味がなさそう。
- 隠された傷: 中学時代、ピアノの天才と呼ばれていた。コンクール前日、親友でありライバルだった少年が交通事故で亡くなった。「あいつの代わりに弾いても意味がない」と全てを捨てた。
- 言えなかった言葉: 「お前がいないと、俺は弾けない」
- 成長アーク:
  1. 拒絶（ランク1-2）: 主人公を鬱陶しがる。「放っとけ」
  2. 無関心の裏（ランク3-4）: 音楽室の前で立ち止まる姿を目撃
  3. 壁（ランク5）: 親友の命日。墓前で初めて感情を爆発させる
  4. 再起（ランク6-8）: 鍵盤に触れ始める。最初は音が出ない
  5. 演奏（ランク9-10）: 学園祭で、親友に捧げる演奏を行う
- 代表台詞:
  - 怒り: 「うるせえ。お前に何がわかる。何もわからねえくせに。」
  - 傷: 「…あいつが死んで、俺の音も死んだんだよ。」
  - 成長: 「弾きたいんだ。もう一度。あいつに聴かせたい曲がある。」
  - 絆: 「お前がいなかったら、俺はずっと屋上で寝てたな。……サンキュ。」
- 戦闘スタイル: 炎属性 + 物理。攻撃力が高いアタッカー。HPが低いほどクリティカル率UP
- コープ報酬: 共鳴体合体時にボーナス経験値 / 蓮の固有スキル「業火の咆哮」解放

**III. 月城 ひなた（つきしろ ひなた）— 星**

- 表の顔: 学園のムードメーカー。SNSフォロワー10万人の人気インフルエンサー。常に笑顔。
- 隠された傷: 両親の離婚係争中。父と母の両方から「味方になれ」と言われ、引き裂かれている。SNSの自分だけが「本当の自分」だと思い込むことで現実から逃避。
- 言えなかった言葉: 「お父さんもお母さんも、私のことは見てないよね」
- 成長アーク:
  1. キラキラ（ランク1-2）: 配信に誘ってくる。表面的だが楽しい
  2. 違和感（ランク3-4）: オフの時の表情が全く違うことに気づく
  3. 壁（ランク5）: 親の裁判の話を聞いてしまう。「見ないで」と叫ぶ

  5. 本当の笑顔（ランク9-10）: 両親に自分の気持ちを伝える
- 代表台詞:
  - 怒り: 「やめてよ、勝手に可哀想とか思わないで！ 私は全然平気だから！」
  - 傷: 「……フォロワーが何万人いても、誰も迎えに来てくれないんだ。」
  - 成長: 「カメラ止めても、私は私でいられるんだね。」
  - 絆: 「ねえ、今度……画面越しじゃなくて、隣で笑ってくれる？」
- 戦闘スタイル: 風属性 + 速度型。連続攻撃と回避に秀でる

### 4.4 ステータス成長システム

**内面能力（5項目）**:

| 能力名 | 意味 | 上げる行動 | 解放コンテンツ |
|---|---|---|---|
| 勇気（Courage） | 困難に立ち向かう力 | 残響界探索・度胸試し・夜間外出 | 蓮の共鳴ランク5以降 / 特定選択肢 |
| 知恵（Wisdom） | 知識と洞察 | 勉強・授業応答・読書 | テスト順位UP / 真澄の共鳴 / 分析精度 |

## 6. ストーリー構成

### 3幕構成

**第1幕: 共鳴の目覚め（4月〜7月）**

- 4月: 蒼葉奏が澄波市に転校。イヤーカフの力で「聞こえない声」が聞こえ始める
- 4月末: 初めての黄昏刻。残響界に引きずり込まれ、共鳴体を覚醒させる
- 5月: 凛・蓮が相次いで共鳴体を覚醒。パーティ結成。最初のダンジョン「沈黙の廊下」攻略
- 6月: ひなたが残響界に迷い込み、覚醒。2番目のダンジョン「虚飾のステージ」攻略
- 7月: 真澄が加入。3番目のダンジョン「空洞の白室」攻略。夏休み前の中ボス戦
- テーマ: 「仲間を知り、自分の力を知る」

**第2幕: 共鳴の深化と亀裂（8月〜12月）**

- 8月: 夏休み。自由行動が増えるが、残響界の異変が拡大。各キャラクターの共鳴が深まる
- 9月: 残響界に「調律者」と名乗る存在が現れる。敵か味方か不明
- 10月: 学園祭。キャラクターイベントのクライマックス。裏では「調律者」の正体に迫る
- 11月: **どんでん返し①**: 「調律者」の正体は、主人公の姉・律の残響。律は1年前に自ら残響界に入り、帰れなくなった。姉は「共鳴石を破壊して残響界を消す」つもりだが、それは澄波市の人々の「言えなかった感情」を全て消去することを意味する
- 12月: パーティ内に対立。「姉を救うか、街を救うか」。クリスマスイベントで関係性の決着
- テーマ: 「絆が深まるほど、選択が重くなる」

**第3幕: 残響の果て（1月〜3月）**

- 1月: 共鳴石が暴走開始。現実世界に残響が漏れ出す。一般人にも「聞こえない声」が聞こえ始め、パニックが起きる
- 2月: **どんでん返し②**: 姉・律が残響界に入った本当の理由は「母が奏に向けていた本音（律より奏が重荷だった）を聞いてしまい、奏を守るためにその残響を消そうとした」。つまり姉は奏のために犠牲になっていた
- 3月初旬: 最終ダンジョン「原初の残響」。共鳴石の中枢へ
- 3月中旬: ラストバトル。姉の残響と対峙
- 3月末: エンディング分岐

- 共鳴石の暴走を止められず、澄波市全体が残響界に飲まれる

- 回収: 11月、調律者がイヤーカフを外した瞬間、姉の顔が現れる

**どんでん返し②: 姉の本当の動機**

- 回収: 姉の残響が語る。「お母さんの残響を聞いたの。奏のことを——。あなたに聞かせたくなかった」

## 8. 自己評価と初日タスク

### 自己評価（ペルソナ比較）

| 項目 | スコア | 改善点 |
|---|---|---|
| キャラクター深度 | 7/10 | 各キャラの「日常の何気ない会話」がまだ不足。設計書レベルではアーク構造は良いが、実際の台詞で感情が動くかは実装次第 |
| システムの面白さ | 7/10 | 共鳴と戦闘の連動は設計上良いが、「共鳴ランクが戦闘ダメージに直結」だけでは単調になる可能性。フュージョンの奥深さが鍵 |
| ストーリーの引力 | 8/10 | どんでん返しの構造は強い。「姉のために犠牲」の真相は普遍的な感情に刺さる。ただし伏線の配置密度を実装で落とさないこと |
| UI/演出構想 | 6/10 | 音波モチーフの設計は良いが、ペルソナ5のUI密度には遠く及ばない。具体的なアニメーション仕様の詰めが必要 |

**ペルソナに勝っている点**: 「言えなかった言葉」というテーマの普遍性。ペルソナが「抑圧された自己」を扱うのに対し、本作は「人と人の間の沈黙」を扱う。より多くのプレイヤーが自分の経験と重ねられる。

**このゲームが最も力を入れるべき点**: 共鳴ランク5（壁）とランク10（解放）の台詞・演出。ここがこのゲームの心臓。一文字たりとも妥協しない。

1. Unityプロジェクト作成 + 基本ディレクトリ構成
