## 1. タイトル・コアコンセプト

**ゲームタイトル**: 残響のアルカディア（Arcadia of Echoes）

**キャッチコピー**: 「本当の自分は、まだ誰にも聞こえていない」

**コアテーマ**: 社会が押し付ける「正しい自分」と、心の奥で叫んでいる「本当の自分」の間で引き裂かれる痛み——その残響（エコー）を聴き取り、自分自身の声を取り戻す物語。

**プレイヤーに最終的に感じさせたい感情**: 「自分の声で生きていい」という静かだが確かな肯定感。涙と共に訪れる解放。

**面白さの公式への回答**:
- 感情移入: 全キャラクターが「自分を偽っている痛み」を持つ → プレイヤー自身の経験と共鳴する
- システム融合: 絆を深めることで戦闘での「共鳴」が発生し、隠された技が解放される
- 時間の希少性: 1年間の学園生活。全ての絆を最大にはできない。選ばなかった関係は変質する
- 演出密度: 残響（エコー）をモチーフにした音と波動の視覚演出が全UIに統一

## 2. 技術スタック

**エンジン**: Unity 2022.3 LTS（安定性・2Dと3Dのハイブリッド表現・JRPGの実績）

**選定理由**:
- C#ネイティブで開発効率が高い
- UniTask + DOTweenによるアニメーション・非同期処理の成熟したエコシステム
- ScriptableObjectベースのデータ駆動設計がJRPGと相性抜群
- 2D演出（UI・会話）と3D空間（フィールド・戦闘）のハイブリッドが容易
- インディーJRPGの豊富な先行事例（Chained Echoes, Sea of Stars等）

**必須パッケージ**:
- UniTask（async/await）
- DOTween Pro（アニメーション・イージング）
- TextMeshPro（テキスト演出）
- Addressables（アセット管理・シーン遷移）

## 3. 世界観・舞台設定

### 舞台

**都市**: 朧崎市（おぼろざきし）
- モデル: 神戸 + 尾道 + 横須賀を融合した架空の港町
- 坂と海と古い商店街が共存する。かつては造船業で栄えたが、今は衰退の影が差す
- 街全体に「聞こえないはずの音」が時折響く——残響現象と呼ばれる都市伝説

**学校**: 天鏡学園（てんきょうがくえん）
- 丘の上に建つ伝統校。海が見渡せる屋上が象徴的なロケーション
- 校訓:「調和を以て尊しと為す」——この校訓自体がゲームのテーマと皮肉に対立する
- 生徒は「模範的であること」を強く求められる風土

### 二重世界

**残響界（ざんきょうかい / Echo Realm）**

- 現実世界の「抑圧された声」が蓄積して生まれた裏世界
- 人々が本音を押し殺すたびに、その感情が残響として沈殿し、残響界を形成する
- 視覚イメージ: 現実の街並みが音波のように歪んでいる。建物は波形（ウェーブフォーム）のように揺らぎ、空は音の周波数を示すスペクトログラムの色彩（深い紫〜鮮やかなシアン）。静寂の場所ほど歪みが激しい
- 残響界の深層に行くほど、抑圧された感情は「変質した獣」（ノイズ）として敵対的に具現化する

**残響界が存在する理由**:
- 朧崎市の地下に眠る古代の「聴神殿」が、人間の感情の波動を吸収・蓄積する装置として機能している
- 数百年分の抑圧感情が臨界を超え、現実と平行する別次元を形成した
- 近年、衰退する街の閉塞感が急増し、残響界が不安定化——現実への侵食が始まっている

**残響界への侵入条件**:
- 「共鳴」できる者だけが侵入可能。主人公は転校初日に偶然「共鳴周波数」に同調し、能力を覚醒する
- 侵入方法: 街の各所にある「共鳴点」（ひび割れのように光る裂け目）に触れることで突入
- 仲間は主人公との「感情的共鳴」を経て侵入能力を獲得する

### 主人公

**名前**: 水無瀬 奏（みなせ かなで） ※デフォルト名・変更可
**年齢**: 17歳（高校2年）
**特殊能力**: 「全周波共鳴（オムニ・レゾナンス）」
- 他者の残影と共鳴し、複数の残影を切り替えて使役できる唯一の存在
- ペルソナのワイルドカードに相当。なぜ主人公だけがこの能力を持つかは、終盤の核心的伏線に直結する

**背景**:
- 両親の離婚により、母方の祖母がいる朧崎市へ転校してくる
- 「どこにいても自分の居場所がない」感覚を抱えている
- 物静かだが、他人の感情に異常なほど敏感（これが共鳴能力の素質）
- 転校初日、屋上で聞こえるはずのない「誰かの叫び声」を聴き取り、物語が始まる

## 4. ゲームシステム設計

### 4.1 時間システム

**1日の構造**:

| 時間帯 | 名称 | 行動ポイント | 主な活動 |
|--------|------|-------------|----------|
| 朝 | 登校 | 0（イベント専用） | 通学イベント・天気予報・仲間との会話 |
| 昼 | 授業 | 0〜1（選択肢） | 授業中の質問応答（ステータスUP）・居眠り選択 |
| 放課後 | 黄昏 | 1 | 絆イベント / 残響界探索 / 部活 / バイト / 街探索 |
| 夜 | 宵闇 | 1 | 自室勉強 / 夜の街 / 絆イベント（特定キャラ） / 残響界（特別条件） |
| 深夜 | 残響時 | 0〜1（条件付き） | 月に数回の特殊時間帯。残響界が最も活性化する |

**重要ルール**:
- 放課後に残響界を探索すると体力を消耗し、夜の行動が制限される
- 絆イベントを優先すると残響界の探索が遅れ、月次デッドラインに追われる
- 「全てはできない」という制約がペルソナと同様の緊張感を生む

**時間経過イベントトリガー（例）**:
1. 5月中旬 — 残響界の最初の「ノイズの王」が現実への侵食を開始（初ボス期限）
2. 7月 — 期末試験（知識ステータスチェック）
3. 8月 — 夏休みイベント（特別な絆イベント解放・海のエピソード）
4. 10月 — 文化祭（全キャラの絆を深める集中イベント）
5. 12月 — 冬の真相イベント（第2幕クライマックス・衝撃の裏切り）
6. 2月 — 最終デッドライン（ラストダンジョン期限）
7. 3月 — エンディング分岐

**月次デッドライン**:
- 各章のボス（ノイズの王）は一定期日までに倒さないとバッドエンド分岐
- 期限が近づくと街の異変が増え、BGMが不穏に変化する演出

### 4.2 戦闘システム

**基本形式**: ターン制コマンドバトル（プレスターン系）

**属性と弱点システム**:

| 属性 | 象徴する感情 | 色 |
|------|------------|-----|
| 共鳴（Resonance） | 共感・理解 | シアン |
| 不協和（Dissonance） | 怒り・拒絶 | 赤紫 |
| 静寂（Silence） | 悲しみ・諦め | 藍色 |
| 咆哮（Roar） | 激情・衝動 | オレンジ |
| 残響（Echo） | 記憶・郷愁 | 金色 |
| 虚無（Void） | 空虚・喪失 | 黒 |
| 物理（Impact） | — | 白 |
| 治癒（Heal） | — | 緑 |

**弱点ヒット時の効果: 「ワンモア・エコー」**
- 弱点を突くと「ワンモア」が発動し、追加行動を得る（ペルソナのPress Turnと同等）
- 弱点ヒット時、画面全体に音波エフェクトが走り、敵がよろめく演出
- SEは「共鳴音」（心地よい和音）が鳴り、プレイヤーに快感を与える

**バトンパス相当: 「エコー・リレー」**
- ワンモア・エコー発動時、追加行動を他の仲間に譲渡できる
- リレーするたびにダメージ倍率が上昇（1.25x → 1.5x → 2.0x）
- 絆ランクが高いほどリレー時のバフ効果が追加される
- 4人全員をリレーで繋ぐと「パーフェクト・エコー」ボーナス（SP回復 + 全能力1ターンUP）

**全体攻撃: 「ユニゾン・ブレイク」**
- 全敵をダウン状態にすると発動可能
- パーティ全員が残影と共鳴し、波形が一つに収束する演出
- カットイン → 全員攻撃 → 波形が弾ける爆発エフェクト
- ダメージ倍率は絆ランクに比例（システムとナラティブの融合）

**主人公の特殊能力: 「全周波共鳴」**
- 複数の残影（ペルソナ相当）を所持・切り替え可能
- 残影は残響界で敵を倒すか、「残影融合」で生成
- 残影融合: 2体以上の残影を掛け合わせて新しい残影を創造（ベルベットルーム相当の施設「反響室」で実行）

## 5. キャラクター設計（パーティメンバー詳細）

### パーティメンバー1: 霧島 凛（きりしま りん）

- **年齢**: 17歳（高校2年・同級生）
- **外見**: 黒髪ロング・常に姿勢が良い・制服は一分の隙もない。ただし爪だけが短く切られている（ストレスで噛む癖の名残）
- **初登場シーン**: 転校初日、教室で完璧な歓迎スピーチをする。しかし主人公だけが、彼女の手が微かに震えているのに気づく
- **表の顔**: 完璧な生徒会長。成績トップ。教師からの信頼は絶大
- **真の傷・抑圧**: 幼少期に両親が離婚。「お前が完璧ないい子なら、パパは帰ってくる」と母に言われ続けた。完璧でなければ存在を許されないという恐怖に支配されている
- **成長アーク（5段階）**:
  1. 完璧な仮面: 隙のない優等生として接する。主人公にもよそよそしい
  2. 亀裂: 生徒会の重圧で初めてミスをする。動揺を隠しきれない

## 6. ストーリー構成

### 3幕構成

**第1幕: 共鳴の目覚め（4月〜7月）**

主人公が朧崎市に転校し、残響界の存在を知る。最初の仲間（陽太）と共に残響界を探索し、基本的な戦闘と絆システムを体験する。

- 4月: 転校・残響界への初侵入・陽太が仲間に
- 5月: 凛が仲間に・最初のノイズの王「沈黙の巨像」戦

- 7月: 期末試験・夏前の中ボス「虚偽の合唱団」戦・第1幕クライマックス

**第2幕: 不協和の深淵（8月〜12月）**

仲間が増え、絆が深まる一方で、残響界の真実に近づく。「黒幕」の存在が示唆される。

- 8月: 夏休み特別編（海イベント・剣士郎覚醒）・自由行動期間

- 10月: 文化祭・全キャラの絆集中イベント
- 11月: 朝比奈が仲間に・残響界のシステム的理解が進む
- 12月: **どんでん返し①** — 影宮零の正体が明らかに。零は残響界の存在「最初のエコー」であり、人間になることを望んでいた。零が暴走し、朧崎市の半分が残響界に飲まれる

**第3幕: 残響の帰結（1月〜3月）**

崩壊する世界を前に、仲間たちは最後の選択を迫られる。

- 1月: 零の救出を決意・最終ダンジョン「聴神殿」への突入開始
- 2月: **どんでん返し②** — 聴神殿の最深部で判明する真実。「全周波共鳴」能力を持つ主人公自身が、聴神殿の次のコアとなるべく運命づけられている。主人公が聴神殿のコアになれば残響界は安定するが、主人公は自我を失う
- 3月: 最終決戦・エンディング

### エンディング分岐

**バッドエンド**: 月次デッドラインに失敗、または重要選択肢で「諦める」を選択。残響界が現実を完全に飲み込む。

**ノーマルエンド**: 主人公がコアとなることを受け入れ、世界を救う。仲間たちは主人公の残響を胸に、それぞれの道を歩む。切ないが、美しい別れ。

**トゥルーエンド（条件: 全パーティメンバーの絆ランク8以上 + 意志力MAX）**: 仲間全員の共鳴が結集し、聴神殿のシステムそのものを書き換える。「抑圧された声を消す」のではなく「全ての声が共存する」世界を実現する。主人公も、零も、全員が生還する。最後のシーンは卒業式の朝——屋上で、全員の「本当の声」が一つになる。

### どんでん返し詳細

**どんでん返し①: 影宮零の正体（12月）**

**どんでん返し②: 主人公の宿命（2月）**
- 伏線: 主人公の「異常な共感能力」/ 祖母が主人公を朧崎に呼んだ本当の理由 / 序盤の祖母の意味深な台詞「お前はここに帰ってきたんだよ」/ 主人公の母が朧崎から逃げた過去
- 回収: 水無瀬家は代々聴神殿の「調律者」の血筋。主人公の母は宿命から逃れるために朧崎を離れた。祖母は衰弱する聴神殿を安定させるため、後継者（主人公）を呼び戻した

## 設計メモ（意思決定の記録）

### 2026-06-05 初期設計決定
- **エンジン選定**: Unity 2022.3 LTSを選択。理由: C#ネイティブ、JRPGの実績豊富、ScriptableObjectによるデータ駆動が強力
- **世界観の方向性**: 「声」と「音」をメタファーの中心に据えた。理由: ペルソナの「仮面」「シャドウ」と差別化しつつ、「本当の自分を表現する」というテーマを音響的イメージで統一できる
- **捨てた案**: ① 時間ループもの（P3の影時間に近すぎる）② 夢の世界（ありきたり）③ SNSとの二重生活（現代的だが視覚的差別化が難しい）
- **属性設計**: 感情ベースの属性にした。理由: 戦闘とストーリーが自然に接続する。「弱点を突く＝相手の感情の急所を突く」というメタファーが機能する
