## 2. 技術スタック

### エンジン選定

**Unity 2022.3 LTS**

**選定理由：**
- UniTask / DOTween の組み合わせによる演出実装が最速
- ScriptableObject ベースのデータ設計がJRPGのコープ/ペルソナ管理と相性が良い
- 2Dと3Dのハイブリッド演出（メニュー2D、フィールド3D）が柔軟に組める
- コミュニティとアセットが豊富でソロ/小チーム開発に現実的

### 二重世界：鏡界（きょうかい）

**現実世界：**
歴史と近代が混在する静かな地方都市。古い神社・路面電車・商店街と、新興の住宅地・工場地帯が共存する。表面は穏やかだが、10年前の列車事故の傷跡が市民の心に深く刻まれている。

**鏡界（異世界）：**
現実都市の「鏡映し」として存在する並行空間。表層は現実の建物が歪んだ鏡像として現れるが、深部へ進むほど、都市と人々の「抑圧された記憶」が具現化した幻想的な空間に変貌する。

- **色彩**：深夜の青みがかった銀色。金属質の冷たさ。鏡面のような床。亀裂から滲み出る深紅。
- **質感**：あらゆる表面が鏡面仕上げ。歩くと自分の影が追いかけてくる。影には歪んだ「本当の表情」が映る。
- **空気感**：無音に近いが、遠くから子供の泣き声や大人の怒鳴り声が断片的に聴こえる。

### 鏡界が存在する理由（作品内論理）

10年前（2016年）、鏡ヶ浦市で列車脱線事故が発生し23名が死亡した。事故原因は鏡ヶ浦鉄道（黒宮財閥経営）のメンテナンス費用削減だったが、政治的圧力により「機械的老朽化による不可抗力」として処理された。

**真実を知る者たちの「抑圧」が、集合的な心理的エネルギーとなって空間に染み出したのが鏡界の正体。**

鏡界の法則：
- 「信じたくないが本当のことを信じた」人間には見えるようになる
- 鏡界内で死ぬと、その人の意識が鏡界に吸収され影（シャドウ）として固定される
- 鏡界が完全に現実を侵食すると、街全体が「事故などなかった」という集合的幻覚の中に閉じ込められる

### 鏡界への侵入条件

- 深夜0時〜深夜3時の間のみ侵入可能（「鏡の時間」）
- 鏡面を媒介に侵入：水たまりの反射・ショウウィンドウの鏡・スマホの黒いロック画面
- 専用のアイテム「鏡の欠片（きょうのかけら）」を所持していると日中でも引き込まれることがある
- 鏡界内は主人公にしか完全には知覚できない（仲間は主人公の「共鳴」越しに存在を共有する）

### 4-2. 戦闘システム

**基本形式：** 1〜3人パーティ + 主人公の、最大4人ターン制コマンドバトル

**属性と弱点システム：**

| 属性名 | 記号 | 弱点の反対属性 | 弱点ヒット時の効果 |
|--------|------|-------------|-----------------|
| 炎（えん） | 🔴 | 氷 | DOWN（次の攻撃で追撃可） |
| 氷（ひょう） | 🔵 | 炎 | DOWN + 凍傷（毎ターンHP減少） |
| 風（ふう） | 🟢 | 雷 | DOWN + 集中力低下（スキルコスト増） |
| 雷（らい） | 🟡 | 風 | DOWN + 麻痺（行動不能1回） |
| 光（こう） | ⚪ | 闇 | DOWN + 混乱（ランダム行動） |
| 闇（あん） | ⚫ | 光 | DOWN + 恐怖（行動スキップ確率UP） |
| 物理（ぶつ） | 🟤 | なし | 強制DOWN（弱点関係なく） |
| 念（ねん/サイキック） | 🟣 | なし | DOWN + 心理ダメージ（MP減少） |
| 核融合（かくゆうごう） | 🔶 | なし | DOWN + 爆発（周囲敵にも余波ダメージ） |
| 呪血（じゅけつ/カース） | ❤️‍🔥 | なし | DOWN + 呪い（被ダメージが増幅） |

**弱点ヒット連鎖：** 同一ターン内に全ての敵がDOWN状態になった際、「共鳴爆発」が発動可能になる。

---

**共鳴爆発（きょうめいばくはつ）— Resonance Burst —**

全敵をDOWNにした後に選択できる必殺技。

**演出フロー：**
1. 画面全体が「鏡面」に変化し、亀裂が走る（DOTweenで0.8秒の割れるアニメ）
2. 各パーティメンバーの共鳴が巨大なシルエットとして召喚され、順番に敵陣に攻撃
3. 主人公の共鳴が最後に現れ、フィニッシュ攻撃（使用中の共鳴によって演出が異なる）
4. 暗転後、「鏡が割れた」画面にダメージ数値が砕けた鏡文字で表示
5. 勝利後スプラッシュ：パーティ全員のシルエットが鏡の前に立ち、背後の鏡に影が映っている

**発動後の効果：** 全敵に高ダメージ + 確率でアイテムドロップ率UP + 共鳴EXP獲得UP

---

**主人公の特殊能力：多重共鳴（たじゅうきょうめい）**

- 戦闘中・戦闘外問わず、保有する共鳴（Resonance）を自由に切り替えられる
- 最大保有数：12体（超えると融合が必要）
- 他パーティメンバーは各自1体の共鳴が固定（絆ランクアップで強化のみ）
- 共鳴の入手方法：
  1. **戦闘取得**：ボスや珍しい敵を倒した際、確率で「共鳴の残像」を受け取る
  2. **絆取得**：絆キャラのランクが上がった際、その人物の心象を元にした共鳴を授かる
  3. **鏡炉融合**：2体の共鳴を素材に、新たな共鳴を錬成（スキルを継承できる）

**鏡炉（きょうろ）— Mirror Forge —**

- 鏡界の最深部近くにある「鏡工房」
- 運営者：磨師（みがきし）/ The Polisher（老婆、正体不明。「磨けば、どんな鏡も真実を映す」が口癖）
- 機能：共鳴融合・スキル継承・共鳴図鑑（コンプリート要素）

### 4-3. 共鳴絆システム（きょうめいきずなシステム）

**システム名称：共鳴絆（Resonance Bond）**

- 最大ランク：10
- 各キャラクターにアルカナ（タロットの大アルカナ）が対応
- ランクアップの条件：そのキャラとの専用イベントを見て、「共鳴選択肢」で正しい応答をすること
  - 共鳴選択肢：キャラの本音に「共鳴（共感・直視・受け入れ）」するか「回避（同情・否定・無視）」するかを選ぶ
  - 共鳴を選ぶとランクUPが早まり、特殊台詞が解放される
  - 回避を選んでもゲームは進むが、ランク10到達が遅くなる

**[1] 七星 詩乃（ななほし しの）｜正義のアルカナ**

- 外見：肩までの黒髪。眼鏡。制服はいつも完璧に整えられている。笑顔が「少し頑張りすぎ」に見える。
- 表の顔：生徒会の会計。成績優秀・品行方正。誰からも「しっかりした子」と言われる。
- 隠された傷：家族が多額の借金を抱えており、大学進学を諦めることを親から「お願い」されている。誰にも言えず、完璧な自分を演じ続けることで「決断を先送り」にしている。
- 成長アーク：
  1. 完璧な仮面 → 2. 主人公に小さな弱音 → 3. 借金の発覚・怒りと悲しみ → 4. 「諦める自分」への怒りが解放 → 5. 自分の人生を選び取る決断
- 代表台詞：
  - **怒り**：「なんで私だけ…。なんで私だけ、諦めなきゃいけないの！」
  - **傷**：「笑っているのが一番楽なの。笑っていれば、何も考えなくて済むから。」
  - **成長**：「借金は払う。でも、人生は返さない。」
  - **絆**：「あなたが見ていてくれるから、私は本当の顔で笑える。」
- 戦闘スタイル：近接物理（刀型の武器）＋風属性。攻守バランス型。デバフスキル多め。
- 絆報酬：ランク5→スキル「正義の一閃」（全体物理・DOWN付与確率UP）、ランク10→共鳴「ジュスティティア」入手・固有パッシブ「真義」（弱点を突かれた際、仲間へのダメージを肩代わりする確率発動）

**[2] 朽木 玲（くちき れい）｜塔のアルカナ**

- 外見：背が高い。制服の第一ボタンを外している。短く切った黒髪。目つきが鋭いが、よく観察すると視線が常に「どこか遠く」を向いている。
- 表の顔：不良キャラ。授業をサボる。教師への反抗。生徒の間では「関わらない方がいい」と噂される。
- 隠された傷：父（黒宮市議会議員・黒宮崇典）が2016年の事故の責任者であることを、玲は高校入学後に知った。以来、父を告発しようとしたが証拠を握りつぶされ、沈黙を「強要」されている。学校でのグレた態度は、誰にも近づいてほしくないという防衛機制。
- 成長アーク：
  1. 孤立した不良 → 2. 主人公との不本意な共闘 → 3. 父の正体が表面化（8月イベント） → 4. 父を告発するか守るかの葛藤・崩壊 → 5. 「父を愛しながらも真実を選ぶ」決断
- 代表台詞：
  - **怒り**：「俺の父親が人を殺した。それが事実だ。他に何が必要だ。」
  - **傷**：「告発したら、誰かが幸せになるのか？死んだ人は戻らないのに。」
  - **成長**：「俺は父親を憎んでいない。だから、言わなきゃいけない。」
  - **絆**：「お前だけには、本当の俺を見せてよかった。」
- 戦闘スタイル：遠距離魔法（炎・核融合）。攻撃特化型。HPが低い。
- 絆報酬：ランク5→スキル「業火」（単体高火力・敵のHP低下で威力増大）、ランク10→共鳴「アグニ」入手・固有パッシブ「告発者」（ボス戦開始時に敵の弱点を1つ自動開示）

**[3] 御影 ことは（みかげ ことは）｜月のアルカナ**

- 回収：8月、主人公が柚子の店で古い新聞記事（事故報道）を見ていると、その裏面に「鏡ヶ浦市議会 黒宮崇典議員」の顔写真がある。その日、玲のスマホに「黒宮家」の表示で電話がかかってくる場面を目撃。すべてが繋がる。
- 感情的インパクト：プレイヤーが「玲のグレた理由」をずっと「反抗期」だと思っていたことを裏切る。父を守るために孤立していた孤独な少年の姿が浮かび上がる。

**どんでん返し②「あの日の生存者」（1月・主人公の真実）**

- 伏線：主人公が転入前の記憶が曖昧 / 母が鏡ヶ浦市の話題を避ける / 鏡村 空（主人公の影）が「君はあの日ここにいた」と繰り返す / 事故の被害者名簿に「月浦」という名前（母の旧姓）がある / ことはが初めて主人公を鏡界で見たとき「あなた、鏡界に匂いがついている。古い匂い」と言う
- 回収：1月、山崎の部屋に残された証拠の中に「事故生存者のリスト（子供）」があり、そこに「月浦 響」という名前がある。母に電話すると沈黙の後「…ごめん、話さなかった」。主人公は幼少期に事故で臨死体験をし、母が「忘れさせるため」に引越し・記憶封印を図ったことが判明。
- 感情的インパクト：プレイヤーが「なぜ主人公だけが鏡界に関われるのか」という疑問への答え。同時に、主人公が「謎を解く探偵」ではなく「謎の一部」だったことへの衝撃。

### 主要画面レイアウト

**タイトル画面：**
- 全画面を使った黒い鏡面。中央にゆっくりとひびが入り、その隙間から金色の文字で「鏡夜のアルカナ」が滲み出てくる
- BGM：アカペラと弦楽器のみ。歌詞は「見えないふりをしてきた / 鏡の前で、今日も」

**フィールド（現実世界）：**
- 3D俯瞰視点（ペルソナ5スタイル）
- 右上：時間帯アイコン（太陽/月のシルエット）+ 残り行動ポイント（鏡の欠片アイコン×N）
- 右下：日付（「鏡ヶ浦市 4月 第1週 月曜日 / 放課後」）
- ミニマップなし（意図的。「直感で歩く」体験を重視）

**フィールド（鏡界）：**
- 現実フィールドと同じ構造だが、シェーダーで全体が鏡面質感に
- BGM：現実BGMを逆再生+エコー処理したもの
- 床に主人公の影が「少し遅れてついてくる」（視覚的な不安感の演出）

**行動選択メニュー：**
- 画面の端から「割れた鏡の破片」がスライドイン
- 各選択肢は鏡の破片の上に表示される
- 選択時：破片が光り、選ばなかった破片が崩れ落ちる（DOTweenで0.3秒）
- カーソル移動音：ガラスが擦れる音（軽く）

**戦闘画面：**
- 画面下半分がUIエリア。鏡面の黒背景。
- HP/SPバーが「亀裂グラフィック」デザイン：HPが減るほどバーに亀裂が入る
- ターン順表示：右端に縦一列で次の行動者のアイコン
- 弱点ヒット時：画面端がその属性カラーで一瞬フラッシュ
- DOWN時：対象の足元に「鏡が割れる」エフェクト

**ステータス画面：**
- ペルソナ5のコンセプトアートスタイル。斜めのカードが積み重なった表示
- 内面ステータスの5項目は五芒星（ペンタグラム）グラフで表示
- 「共鳴図鑑」タブ：所持/未所持の共鳴が鏡に映ったシルエットで表示

**共鳴絆（社会リンク）画面：**
- キャラクターのポラロイド写真がコルクボード風に並ぶ
- ランクが上がるほど写真が「現像されていく」（最初はセピア/ぼけぼけ → ランク10で鮮明なカラー写真に）
- 各キャラのアルカナカードが横に並ぶ
- BGM：そのキャラのテーマのアコースティックアレンジ

## 8. 自己評価と初日タスク

### ペルソナシリーズとの比較自己評価

| 項目 | スコア | 改善点 |
|------|--------|--------|
| キャラクター深度 | **8/10** | 玲と詩乃は深いが、奈央・蒼太の傷の普遍性がやや弱い。実装段階でセリフ密度を上げる必要がある |
| システムの面白さ | **7/10** | 鏡炉（Resonance Fusion）の戦略深度を上げる必要がある。属性10種は多すぎる可能性→テスト実装で絞る判断 |
| ストーリーの引力 | **9/10** | 2つの伏線回収は強力。「事故」という具体的な社会的テーマがテーマと深く噛み合っている |
| UI/演出構想 | **8/10** | 鏡のモチーフとテーマの一体感は強い。ただし「鏡面エフェクト」の技術的実装コストが高く、スコープ管理が必要 |

**ペルソナシリーズに勝っている点：**
- テーマ（直視できない真実）とシステム（鏡界・共鳴）の哲学的一致度が高い
- 事故という社会的テーマが、個人の心理的テーマと二重構造で機能している（個人の抑圧 ＝ 社会の抑圧）
- 「鏡村 空」という主人公の影キャラが、ゲーム全体の謎とテーマを体現するユニークな設計

**このゲームが最も力を入れるべき点：**
> 絆イベントのセリフ密度と感情的精度。キャラクターが「言いたくないことを言ってしまう瞬間」の演出に、開発リソースの30%以上を投じること。システムよりナラティブが先。
