## 1. タイトル・コアコンセプト

**ゲームタイトル**: 残響のアストラ / Astra of Echoes
**キャッチコピー**: 「忘れた声が、世界を壊す」
**コアテーマ**: 人は「聞こえないふりをした本音」に蝕まれる——抑圧された感情が異世界として実体化する世界で、他者の本音に向き合い、自分自身の沈黙と対峙する物語。
**プレイヤーに最終的に感じさせたい感情**: 「自分も誰かの声を聞こえないふりしていたかもしれない」という静かな後悔と、それでも明日から耳を澄ませようと思える温かさ。

### コアテーマの設計原則
- 全てのシステムは「声を聴く/聴かない」のメタファーで統一する
- 社会リンク = 相手の「残響（聞こえないふりをされた声）」に耳を傾ける行為
- 戦闘 = 抑圧された感情が暴走した姿と向き合う行為
- 時間の有限性 = 全員の声は聴けない。誰の声を優先するかという選択の重さ

## 3. 世界観・舞台設定

### 舞台
**都市**: 静岡県・凪咲市（なぎさし）— 海沿いの中規模都市。穏やかな表層の下に、住民たちの「沈黙」が堆積している。
**学校**: 県立 潮鳴高等学校（しおなりこうとうがっこう）— 主人公が2年生として転入する。

### 二重世界

**現実世界**: 凪咲市での高校生活。授業、部活、バイト、友人関係。一見穏やかだが、誰もが「本音」を飲み込んでいる。

**異世界「残響界（ざんきょうかい）」**: 人々が飲み込んだ本音——「言えなかった言葉」「聞こえないふりをされた叫び」——が実体化した異空間。凪咲市と同じ地理構造を持つが、抑圧された感情によって風景が歪んでいる。
- **視覚イメージ**: ガラスのように透明だが罅（ひび）の入った建物。空気中に浮遊する文字の断片（誰かの飲み込んだ言葉）。色彩は人物ごとに異なり、その人の抑圧の性質を反映する。例：怒りを抑圧した人のエリアは赤黒い、悲しみは深い藍色。
- **音**: 常に微かなノイズが聞こえる。それは「聞こえないふりをされた声」の残響。

### 異世界の存在理由
凪咲市の地下に眠る「共鳴石（きょうめいせき）」——古来より人の感情に共振する鉱石——が、現代社会の抑圧の蓄積に反応し、臨界点を超えて「残響界」を生成した。抑圧が一定量を超えると「残響界」が現実を侵食し始め、影響を受けた人は「沈黙症（ちんもくしょう）」に陥る（感情を一切表現できなくなる）。

### 侵入条件・方法
主人公のスマートフォンに突如インストールされたアプリ「ECHO」。このアプリは共鳴石と主人公の特殊な共感能力が共振して生まれたもの。沈黙症の患者の近くでアプリを起動すると、その人の「残響」に入ることができる。

### 主人公
**名前**: 響 蓮（ひびき れん）※プレイヤーが変更可能
**年齢**: 17歳（高2）
**背景**: 幼少期に母親が突然失踪。母は常に「大丈夫」と言い続け、SOSを出さなかった。母の声を「聞こえないふり」してしまった——という罪悪感を抱えている。この原体験が共鳴石と共振し、他者の「残響」を聴く力を得た。
**特殊能力「共鳴（レゾナンス）」**: 他者の抑圧された感情に共鳴し、その感情を「ペルソナ」のように戦闘で使役する力。複数の残響を宿すことができる（ワイルドカード相当）。
**外見**: 黒髪、常にイヤホンを片耳だけつけている（聴く/聴かないの象徴）。

## 4. ゲームシステム設計

### 4-1. 時間システム

**1日の構造**:
| 時間帯 | 行動ポイント | 主な行動 |
|--------|-------------|---------|
| 朝 | 0（イベントのみ） | 通学イベント、朝の会話 |
| 授業 | 0（イベント+選択肢） | 授業中の質問応答→知性UP |
| 昼休み | 1 | 屋上・教室での会話（共鳴リンク進行） |
| 放課後 | 1 | 部活 / バイト / 共鳴リンク / 残響界探索 / 自由行動 |
| 夜 | 1 | 自室で勉強 / 夜の街で活動 / 共鳴リンク（特定キャラ） |
| 深夜 | 0（条件付き） | 胆力が一定以上で行動可能（特殊イベント） |

**時間経過イベントトリガー（例）**:
1. **沈黙症発生（月1回）**: 新たなNPCが沈黙症に陥り、期限内に残響界を攻略しなければ永久に感情を失う
2. **季節イベント**: 文化祭（10月）、クリスマス（12月）、バレンタイン（2月）——共鳴リンクキャラとの特別会話
3. **中間・期末テスト**: 知性ステータスが試される。高得点で社会的評価UP→特定リンク解放
4. **満月の夜**: 残響界が活性化し、通常入れないエリアが開放される（月1回の特殊探索）
5. **母の手がかり**: 不定期に母の失踪に関する手がかりが見つかる（メインストーリー進行）
6. **共鳴暴走**: 主人公の共鳴力が不安定になるイベント（感情の蓄積による）

**月次デッドライン**:
各月に1人の沈黙症患者が発生。その月の月末までに残響界のボスを倒さなければ、そのNPCは永久に沈黙症のまま（関連する共鳴リンクが閉じる）。ペルソナ3のタルタロス+フルムーンボスの構造。

### 4-2. 戦闘システム

**基本形式**: ターン制コマンドバトル（プレスターン相当の独自システム）

**属性システム「感情属性」**:
| 属性 | 象徴する感情 | 色 |
|------|-------------|-----|
| 炎（イラ） | 怒り | 赤 |
| 氷（トリス） | 悲しみ | 藍 |
| 雷（フルゴ） | 恐怖 | 紫 |
| 風（ヴェント） | 焦り | 緑 |
| 光（ルーチェ） | 希望 | 金 |
| 闇（オンブラ） | 絶望 | 黒 |
| 物理 | — | 白 |
| 万能 | — | 虹 |

**弱点ヒット時の効果「感情共鳴（エモーショナル・レゾナンス）」**:
- 弱点を突くと「共鳴ゲージ」が上昇
- 共鳴ゲージMAXで**「共鳴一斉攻撃（レゾナンス・バースト）」**が発動可能
- レゾナンス・バースト: パーティ全員がスタイリッシュなカットインと共に一斉攻撃。共鳴リンクのランクが高いキャラほどダメージ倍率UP（システムとナラティブの融合）

**プレスターン相当「エコーターン」**:
- 弱点ヒットまたはクリティカルで「エコー（追加行動権）」を1つ獲得
- エコーは即座に使うか、ストックして後から連続行動に使える（最大3ストック）
- ミス or 耐性で逆にエコーを1つ失う
- これによりペルソナ3〜5の「1MORE」の爽快感を再現しつつ、ストック要素で戦略性を追加

**主人公の特殊能力「共鳴切替（レゾナンス・シフト）」**:
- 残響界で倒した敵の感情を「残響」として吸収し、戦闘で使役できる
- 複数の残響を所持し、ターン中に切り替え可能（ワイルドカード相当）
- 残響同士を融合して新たな残響を生み出す「残響融合（エコー・フュージョン）」

## 5. キャラクター設計（パーティメンバー）

### パーティメンバー1: 氷室 楓（ひむろ かえで）

**年齢**: 17歳（高2・同級生）
**外見**: 長い黒髪をハーフアップにまとめた端正な顔立ち。制服は常に完璧に着こなす。唯一、左手首に隠したミサンガだけが「自分らしさ」の象徴。
**初登場シーン**: 転入初日、クラスメイトに囲まれる主人公を自然にフォローする。完璧な笑顔の裏で、一瞬だけ表情が消える瞬間をプレイヤーだけが目撃する。

**表の顔**: 成績優秀、容姿端麗、コミュニケーション能力抜群。「凪咲の才媛」と呼ばれる完璧な優等生。
**真の傷**: 医師の両親から「完璧であること」以外を認められずに育った。感情を見せると「みっともない」と叱られた幼少期。本当の自分が何を好きなのか、何をしたいのか、自分でもわからなくなっている。
**成長アーク**:
1. 完璧な仮面を被った状態で主人公と出会う
2. 主人公の前でだけ、少しずつ仮面がずれる
3. 残響界で「完璧な自分」の虚像と対峙する
4. 仮面を脱いだ「空っぽの自分」に絶望する
5. 「空っぽでも、今からここに中身を入れていけばいい」と自分を受容する

**代表台詞**:
- 怒り: 「……私の何を知ってるの？ 完璧な氷室楓しか見てないくせに」
- 傷: 「お父さんもお母さんも、"楓"が好きなんじゃない。"完璧な子供"が好きなの」
- 成長: 「今日の私はたぶん、昨日より少しだけ……私、だと思う」
- 絆: 「あなたの前では、点数のつかない私でいられる。……ありがとう」

**戦闘スタイル**: 氷属性（トリス）+ 回復。冷静な判断力を反映した補助・回復特化。覚醒後は攻撃的な氷魔法も習得。

**第1幕「沈黙の街」（4月〜7月）**:

**第3幕「残響の果て」（1月〜3月）**:
黒幕は共鳴石そのもの——人類の抑圧の総体が意志を持った存在「大いなる沈黙（グランド・サイレンス）」。全ての人間の感情を「沈黙」させることで苦しみを消そうとしている。最終決戦は主人公自身の残響界——母への未練と罪悪感の中で、「聴くこと」の本当の意味を見出す。

### エンディング分岐

**バッドエンド「完全なる沈黙」**:
共鳴リンクを十分に育てず、仲間との絆が弱い状態で最終決戦に臨む。主人公は「大いなる沈黙」に飲まれ、自ら声を手放す。「聴かなければ、傷つかない」というメッセージで終わる。条件: 共鳴リンク平均ランク4以下。

**ノーマルエンド「それでも、聴こえる」**:
最終ボスを倒すが、共鳴石は破壊されず残響界は存続。主人公は凪咲市を去るが、仲間たちはそれぞれの「声」を持って生きていく。母の真相は不明のまま。条件: 共鳴リンク平均ランク5〜7、共感ステータス3以下。

**トゥルーエンド「残響のアストラ」**:
全ての共鳴リンクを十分に育て、共感ステータス最大で最終決戦に臨む。主人公は母の残響を発見し、母が実は主人公を守るために自ら沈黙を選んだことを知る。「大いなる沈黙」を倒すのではなく、「聴く」ことで浄化する。共鳴石は「人の声を増幅する石」として生まれ変わる。主人公は凪咲市に残り、仲間と共に歩み続ける。最後の台詞:「聴こえるよ。全部、聴こえてる」。条件: 共鳴リンク平均ランク8以上、共感ステータス最大、母の手がかり全回収。

### どんでん返し

**どんでん返し1「共鳴石の真実」**:
共鳴石は「人の声を抑圧する装置」ではなく、元々は「人の声を増幅する装置」だった。人類が互いの声を聴くことに耐えられなくなった結果、石が抑圧のエネルギーで汚染され、残響界を生成するようになった。つまり、元凶は石ではなく「聴くことを放棄した人間そのもの」。

**伏線**: 序盤で主人公が触れた共鳴石の欠片が、「温かい」と感じる描写。残響界の最深部で流れる音が「歌」に聞こえるという詩織の発言（6月）。

**どんでん返し2「母の沈黙」**:
主人公の母は失踪したのではなく、自ら残響界に入り、「大いなる沈黙」の進行を食い止めていた。母は共鳴の力を持つ初代の使い手であり、主人公を守るために自分の存在を世界から「消音」した。主人公が母の声を「聞こえないふり」していたのではなく、母が「聴こえないように」していた。

**伏線**: ECHOアプリに時々表示されるバグのような文字列が、実は母のメッセージ（4月から断片的に登場）。主人公の片耳イヤホンの癖が、実は幼少期に母が片耳だけイヤホンをしていた姿の模倣（7月に回想で示唆）。
