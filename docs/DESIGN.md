## 1. タイトル・コアコンセプト

**ゲームタイトル**: 逆光のアリア（Aria of Backlight）

**キャッチコピー**: 「その光の裏側に、あなたの本当の歌がある。」

**コアテーマ**:
誰もが光の裏側に影を持つ。その影を見つめる勇気だけが、本当の繋がりを生む。

**プレイヤーに最終的に感じさせたい感情**:
「自分の弱さを受け入れることは、恥ではなく強さだった」という深い安堵と、
隣にいてくれた仲間への静かな感謝。エンディングで涙が止まらなくなる体験。

**「逆光」の意味**:
逆光（contre-jour）の中では、相手の顔は見えない。シルエットしか見えない。
私たちは日常で、他人の「シルエット」しか見ていない。
このゲームは、逆光の向こう側にある本当の顔を見に行く物語。

**「アリア」の意味**:
アリアはオペラの独唱曲。一人一人が持つ「本当の声」の象徴。
社会が求める「合唱」に紛れて、自分だけのアリアを歌えなくなった人々を救う。

## 3. 世界観・舞台設定

### 舞台

**都市**: 逆瀬川市（さかせがわし）
兵庫県の架空の都市。阪急沿線の住宅街と古い商店街が共存する街。
丘の上に建つ学園から見下ろす夕景が美しいが、その美しさの裏で
住民たちは「言えない本音」を抱えて生きている。

**学校**: 私立 暁星学園（ぎょうせいがくえん）
丘の上に建つ中高一貫校。校訓は「和光同塵（わこうどうじん）」
——自分の光を隠し、世の中に溶け込めという教え。
皮肉にも、この校訓が生徒たちの本音を抑圧する装置になっている。

**季節**: 4月（転校）〜翌3月（卒業）の1年間

### 二重世界

**現実世界**: 逆瀬川市での日常。学校、商店街、住宅街、神社、河川敷。
普通の高校生活だが、「みんな何かを隠している」という微かな違和感がある。

**裏面世界（リメン）**:
人間の抑圧された感情が臨界点を超えたとき、現実の裏側に「裏面」が開く。
裏面は、**その人が「こうであってほしい」と願う理想化された世界**。

**重要な特徴**: 裏面は最初、美しく見える。
- 暗いダンジョンではなく、眩しいほどの光に満ちた「理想郷」
- その人が望む完璧な世界——だが、よく見ると歪んでいる
- 美しい檻。理想に囚われた人間の心象風景
- 奥に進むほど、理想の裏にある「本当の感情」（怒り、悲しみ、恐怖）が漏れ出す
- 最深部に「裏面の主」——その人の抑圧された本音が具現化した存在がいる

**なぜ裏面は存在するのか**:
逆瀬川市の地下には古代の「鏡石」が埋まっている。
この鏡石は人間の感情を映し、増幅する性質を持つ。
かつては神事に使われていたが、都市開発で地中に封じられた。
抑圧された感情のエネルギーが鏡石に蓄積し、裏面として現実に侵食し始めている。

**裏面への侵入方法**:
街の各所にある「鏡面」（ガラス、水面、鏡など反射面）から侵入できる。
主人公と仲間のみが「逆光の時刻」——夕暮れの一瞬、光が完全に逆転する瞬間——に
鏡面を通って裏面に入れる。

**裏面の視覚的イメージ**:
- 基調色: パールホワイト + 淡いプリズム色（理想の美しさ）
- 奥に進むと: 色が滲み、ノイズが走り、暗い色が混じる
- 最深部: 完全に色が反転し、影だけの世界になる
- 質感: 磨りガラスの向こう側のような、「見えそうで見えない」感覚

### 主人公

**名前**: 朝霧 奏（あさぎり かなで）——デフォルト名、変更可能
**年齢**: 17歳（高校2年生）
**外見**: 少しくせのある黒髪、左目の下に小さなほくろ。穏やかだが、どこか影のある表情。

**背景**:
幼少期に母親を亡くしている。母は音楽家だった。
母の死後、父と二人で各地を転々とし、逆瀬川市に転校してくる。
「誰にも深入りしない」ことで自分を守ってきた。

**特殊能力**: 「逆光視（ぎゃっこうし）」
逆光の中で人を見たとき、その人の「影」——抑圧された本音——が一瞬見える能力。
母の形見のイヤリング（片耳だけ）が触媒。
この能力により裏面世界を知覚し、侵入できる。

**主人公の「共鳴体」**: オルフェウス・リバース
冥界から妻を連れ戻そうとしたオルフェウスの「振り返ってしまった」後悔が核。
「大切な人を失う恐怖」と「それでも繋がろうとする意志」の二面性を持つ。

## 4. ゲームシステム設計

### 4-1. 時間システム

**1日の構造**:

| 時間帯 | 行動枠 | できること |
|--------|--------|-----------|
| 朝 | 自動 | 通学イベント・天候確認・仲間との会話 |
| 授業 | 選択1回 | 授業に集中（知識↑）/ 居眠り（体力回復）/ 内職（器用さ↑）|
| 昼休み | 選択1回 | 仲間と昼食（絆↑）/ 図書室（知識↑）/ 購買部（アイテム購入）|
| 放課後 | 選択2回 | 部活 / 絆イベント / 裏面探索 / 街探索 / バイト |
| 夜 | 選択1回 | 自室で勉強（知識↑）/ 読書（感性↑）/ 仲間に電話（絆↑）/ 夜の街探索 |
| 深夜 | 条件付き | 度胸が一定以上で夜の街に出られる。特殊イベント発生 |

**時間経過イベントトリガー（例）**:
1. 5月第2週: 最初の裏面が発生（チュートリアルダンジョン）
2. 6月末: 期末テスト（知識チェック・クラス順位）
3. 8月: 夏休み——自由時間が増えるが、裏面の活動も活発化
4. 10月: 文化祭——仲間全員との特殊イベント・告白チャンス
5. 12月: クリスマスイベント——絆レベルに応じた分岐
6. 1月: 裏面の侵食が現実世界に漏れ始める
7. 2月: 真相解明イベント——鏡石の秘密
8. 3月: 最終決戦 → エンディング分岐

**月次デッドライン**:
各章（約1〜2ヶ月）ごとに「裏面の主」が覚醒する期限がある。
期限までに裏面を攻略しないと、対象の人物が「理想の殻」に完全に閉じこもり、
バッドイベントが発生する（取り返しのつかないペナルティ）。

### 4-2. 戦闘システム

**基本形式**: ターン制プレスターンコマンドバトル

**属性システム**:

| 属性 | 象徴する感情 | 色 |
|------|-------------|-----|
| 焔（ほむら）| 怒り・情熱 | 赤 |
| 氷結（ひょうけつ）| 孤独・拒絶 | 青 |
| 雷鳴（らいめい）| 衝動・焦燥 | 黄 |
| 疾風（はやて）| 逃避・自由 | 緑 |
| 光輝（こうき）| 希望・理想 | 白 |
| 深淵（しんえん）| 絶望・諦め | 黒 |
| 共鳴（きょうめい）| 繋がり・共感 | 虹色 |

**弱点ヒット時の効果**:
- 「共鳴ターン」獲得（1MORE相当）——追加行動が可能
- 敵がよろめき状態になる
- 画面が一瞬フラッシュし、BGMにアクセントが入る演出

**全員ダウン時**: 「カタルシスブレイク」（All-out Attack相当）
名称: カタルシスブレイク
演出イメージ:
- 画面が白く飛ぶ → パーティ全員のシルエットが逆光で浮かぶ
- 一斉攻撃のカットイン（各キャラの目元アップ → 決めポーズ）
- フィニッシュ時に水彩画風のスプラッシュエフェクト
- 決め台詞がランダムで表示される

**主人公の特殊能力**: 「共鳴切替（レゾナンス・シフト）」
主人公のみ、戦闘中に複数の共鳴体を切り替えられる。
仲間はそれぞれ固有の共鳴体1体を持つ。
共鳴体は「絆」のランクに応じて進化する。

**共鳴体の入手方法**:
- 裏面世界で敵（裏面の残滓「エコー」）を説得して吸収
- 「合奏」——2体の共鳴体を融合して新しい共鳴体を生み出す（フュージョン相当）

### 4-3. 絆システム（社会リンク / コープ相当）

**システム名称**: 「調律（ちょうりつ）」

由来: 楽器の調律のように、人と人の「音」を合わせていく行為。
他者との絆を深めることを「調律する」と表現する。

**最大ランク**: 10（各キャラクター）

**調律ランク報酬**:
- ランク上昇時に対応する共鳴体にボーナス経験値
- 特定ランクで専用スキル解放
- ランク10到達で共鳴体が最終進化
- 戦闘中の特殊連携技が解放される

**調律キャラクター一覧（8人 + 特殊枠）**:

#### 1. 皐月 遥（さつき はるか）—— I. 魔術師
- 表の顔: 明るく社交的な学級委員長。誰にでも笑顔を向ける「太陽」
- 隠された傷: 「良い子」でいなければ愛されないという強迫観念。母親からの条件付き愛情
- 成長アーク: 他人の期待に応える自分 → 期待を裏切る恐怖 → 「嫌われても私は私」
- 代表台詞:
  - 「大丈夫、私に任せて！　……大丈夫だから。」（表の声、最後だけ小さくなる）
  - 「ねえ、もし私が『いい子』じゃなくなったら……まだ隣にいてくれる？」（傷）
  - 「もう誰かのための笑顔はやめる。これは、私が笑いたいから笑ってるの。」（成長）
- コープ報酬: 戦闘中「鼓舞」——パーティ全体の次ターン攻撃力UP

#### 2. 氷室 蓮（ひむろ れん）—— III. 女帝（男性だが逆位置の象徴）
- 表の顔: 寡黙で成績優秀な副委員長。感情を見せない「氷の壁」
- 隠された傷: 幼少期のネグレクト。「感情を持つと傷つく」と学習した防衛機制
- 成長アーク: 無感情の仮面 → 怒りの爆発 → 「感じることは弱さじゃない」
- 代表台詞:
  - 「感情は非効率だ。論理で十分だろう。」（仮面）
  - 「……うるさい。お前に何がわかる。お前に、何が。」（怒り、声が震える）
  - 「不器用でもいい。俺は、俺の言葉で伝えたいことがある。」（成長）
- コープ報酬: 戦闘中「冷静分析」——敵の弱点を自動で表示

#### 3. 天音 いろは（あまね いろは）—— VI. 恋人
- 表の顔: 美術部の天才。独特の感性で周囲から「変わった子」と見られる
- 隠された傷: 共感覚の持ち主。他人の感情が「色」として見えてしまい、人混みが苦痛
- 成長アーク: 世界が怖い → 自分の「異質さ」を受容 → 「この目があるから見える景色がある」
- 代表台詞:
  - 「……あなたの色、今日はすごく綺麗。でも少しだけ、滲んでる。」（日常）
  - 「私、おかしいんでしょ？　みんなには見えないものが見えるって、病気でしょ？」（傷）
  - 「この世界は、私にしか描けない色で溢れてる。もう怖くない。」（成長）
- コープ報酬: 裏面探索中「色彩感知」——隠し通路・アイテムを可視化

#### 4. 桐生 岳（きりゅう がく）—— VII. 戦車
- 表の顔: 体育会系の熱血漢。剣道部主将。正義感が強い
- 隠された傷: 父親（元警察官）のDVを目撃して育った。「強くなければ守れない」という脅迫
- 成長アーク: 力への執着 → 守れなかった過去 → 「本当の強さは、助けを求められること」
- 代表台詞:
  - 「俺が守る。絶対に。」（表の決意、しかし目が泳いでいる）
  - 「強くなきゃ……強くなきゃ、また誰かが……っ」（過去のフラッシュバック）
  - 「一人で抱え込むのが強さだって思ってた。馬鹿だよな、俺。」（成長）
- コープ報酬: 戦闘中「先制防御」——味方への致死攻撃を自動でかばう

- 成長アーク: 画面の向こう側の安全地帯 → 現実の一歩 → 「画面の外にも味方はいる」

## 6. ストーリー構成

### 3幕構成

**第1幕: 覚醒（4月〜7月）——「影を見る目」**

4月: 転校・日常の始まり。逆瀬川市の「違和感」に気づき始める。
5月: 最初の裏面発生（皐月遥のクラスメイト）。チュートリアルダンジョン。
     主人公の共鳴体が覚醒。皐月が仲間に。
6月: 氷室の裏面発生。感情を持たない「完璧な世界」。
     氷室が仲間に加わる。期末テスト。
7月: 天音の裏面が発生しかける。夏休み前の緊張。
     鏡石の存在を示唆するヒントが出る。

**第2幕: 深化（8月〜12月）——「光の裏側」**

8月: 夏休み。天音の裏面攻略。桐生が仲間に。
     自由時間が多いが裏面の発生頻度も上がる。
9月: 学園内で不穏な噂。「鏡を見ると別の自分が映る」
10月: 文化祭。仲間との絆が深まるイベント。
      だが文化祭の裏で、生徒会長・藤堂美月の裏面が発生。
11月: 藤堂美月の裏面攻略。「完璧な指導者」の仮面の裏。
      鏡石の研究施設の存在が判明。神楽との出会い。
12月: クリスマスイベント。束の間の平穏。
      だが年末、裏面が現実に「漏出」する事件が発生。

**第3幕: 共鳴（1月〜3月）——「逆光の向こうへ」**

1月: 裏面の侵食が加速。一般人にも裏面が見え始める。
     黒幕の存在が明らかになる。
2月: 真相解明——どんでん返し①②が発生。
     最終ダンジョンへの道が開く。
3月: 最終決戦。エンディング分岐。卒業式。

### どんでん返し

**どんでん返し①: 神楽の真意**
伏線: 神楽は「裏面を消す方法を研究している」と言っていた。
回収: 実は神楽は「裏面を安定化させ、妻がいる向こう側の世界を永続化する」ことを目論んでいた。主人公たちの裏面攻略は、彼にとって「邪魔な不安定要素の排除」に利用されていた。
感情的意図: 信頼していた導き手の裏切り。しかし彼の動機は「愛する人に会いたい」という純粋な感情。善悪では割り切れない。

**どんでん返し②: 主人公自身の裏面**
伏線: 主人公だけは裏面が発生しない。「特別な存在」だと思われていた。
回収: 実は逆瀬川市全体が主人公の裏面だった。母を失った主人公が無意識に作り出した「繋がりのある理想の世界」。転校してきたのではなく、鏡石の力で「理想の日常」を創り出していた。
感情的意図: 全ての絆が「作り物」かもしれないという絶望。しかし最終的に——
「たとえ始まりが嘘でも、一緒に過ごした時間は本物だ」という結論に至る。

### エンディング分岐

**バッドエンド**: 真実を受け入れられず、裏面を維持し続ける。「偽りの日常」に戻る。
条件: 最終選択で「このままでいい」を選ぶ。

**ノーマルエンド**: 裏面を消去し、現実に戻る。仲間との記憶は残るが、日常は元通り。
条件: 最終選択で「元の世界に戻る」を選ぶ。

**トゥルーエンド**: 裏面と現実を「調律」し、両方の世界を共存させる。
条件: 全パーティメンバーの絆ランク10 + 特定のフラグ回収。
演出: 卒業式で、仲間全員が主人公に向かって「自分だけのアリア」を歌う。
主人公が初めて泣く。エンドロール後、母の形見のイヤリングが両耳分になっている。

## 8. 自己評価と初日タスク

### 自己評価（ペルソナ比較）

| 項目 | スコア | 改善点 |
|------|--------|--------|
| キャラクター深度 | 7/10 | 8人の絆キャラは骨格が良い。ただし台詞の「声」がまだ見えない。実装時に声に出して読んで磨く必要あり |
| システムの面白さ | 6/10 | 属性を「感情」に紐づけた点は独自性あり。ただしプレスターン+調律の連携が具体性に欠ける |
| ストーリーの引力 | 8/10 | どんでん返し②「主人公自身が裏面の創造者」は衝撃力がある。ただし伏線の仕込みが不十分 |
| UI/演出構想 | 7/10 | 音楽モチーフの一貫性は良い。ペルソナ5のUIに比べるとまだ大人しい。もっと攻めるべき |

**ペルソナに勝っている点**:
- 異世界が「美しい」という逆転の発想。恐怖ではなく「美しい檻」という新しいダンジョンの感触
- 属性が感情に対応している点。戦闘が「感情をぶつけ合う」行為になる
- どんでん返し②の構造。プレイヤーが「自分が作り物の世界の住人だった」のではなく「自分が世界を作っていた」

**このゲームが最も力を入れるべき点**:
台詞の品質。ペルソナの台詞は「声に出して読みたくなる」レベル。
現状の設計はキャラクターの「構造」は良いが「声」がまだ弱い。
実装フェーズで台詞を一つずつ磨き込むことが最重要。
