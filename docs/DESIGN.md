## 1. タイトル・コアコンセプト

**ゲームタイトル**: 残響のアルカディア（ざんきょうのアルカディア）
**英語タイトル**: Echoes of Arcadia

**キャッチコピー**: 「伝えられなかった言葉が、世界を歪める――」

**コアテーマ**:
> 人が飲み込んだ言葉は消えない。それは"残響"となって世界を歪め、やがて本人すら蝕む。
> 声にすることでしか、人は自分を取り戻せない。

**プレイヤーに最終的に感じさせたい感情**:
「自分にも、ずっと言えなかった言葉がある」という気づき。
そしてエンディングを迎えた後、現実世界で誰かに本音を伝えたくなる衝動。
ペルソナ5が「反逆の意志」を灯したように、このゲームは「伝える勇気」を灯す。

## 3. 世界観・舞台設定

### 舞台

**都市**: 潮鳴市（しおなりし）
神奈川県の架空の海沿いの街。鎌倉と湘南をベースに、坂道と潮風の街。
古い漁港と新しい商業エリアが混在し、どこからでも海が見える。
夕暮れの美しさで知られ、「日本で最も美しい黄昏の街」と呼ばれている。

**学校**: 潮鳴学園（しおなりがくえん）
海を見下ろす丘の上に建つ私立高校。芸術コースが有名で、音楽・美術・演劇の才能が集まる。
しかしその華やかさの裏で、「表現者であれ」というプレッシャーが生徒を蝕んでいる。

### 二重世界

**現実世界**: 潮鳴市の日常。学校、商店街、海岸、図書館、喫茶店。
穏やかだが、人々は本音を飲み込んで生きている。

**残響界（ざんきょうかい / Echo Realm）**:
人々が飲み込んだ言葉が「残響」となって堆積した異世界。
現実の場所と対応しているが、飲み込まれた感情によって歪んでいる。

**視覚的イメージ**:
- 基調色: 深い藍色と透明な青。水中のような浮遊感。
- 質感: 音の波紋が空間を走り、言葉の断片が文字として浮遊している。
- 空気感: 静かだが、遠くから誰かの声が聞こえるような不安感。
- 歪みの強い場所は色が濁り、音が不協和音になる。
- 各ダンジョンは「飲み込んだ感情」に応じたテーマを持つ
  - 例: 怒りを飲み込んだ場所 → 赤黒い炎と割れたガラスの空間
  - 例: 悲しみを飲み込んだ場所 → 止まない雨と沈んだ街並み

**残響界が存在する理由**:
潮鳴市には古くから「言霊」の力が強い土地とされる伝承がある。
飲み込まれた言葉は消えず、地下の霊脈に「残響」として蓄積される。
残響が臨界点を超えると「歪み」が発生し、現実の人間の精神を侵食し始める。
侵食された人は徐々に無気力になり、最終的に「自分の言葉」を完全に失う。

**侵入条件・方法**:
- 主人公の持つスマートフォンのアプリ「Reverb」（残響を感知するアプリが突然インストールされる）
- 残響の歪みが強い場所でアプリを起動すると、残響界への「亀裂」が開く
- 亀裂は特定の時間帯（放課後・夜）にのみ安定して開ける
- パーティメンバーは主人公と「共鳴」している者のみ同行可能

### 主人公

**名前**: 水瀬 蓮（みなせ れん） ※プレイヤーが変更可能
**年齢**: 17歳（高校2年）
**外見**: 黒髪をやや長めに、右目を隠すように流している。穏やかだが、どこか壁を感じさせる表情。ヘッドフォンを常に首にかけている。

**背景**:
前の学校で、親友が深刻な悩みを抱えていた。蓮はそれに気づいていたが、「余計なことを言って関係を壊したくない」と沈黙した。結果、親友は不登校になり、転校した。蓮はその罪悪感から潮鳴市の叔母の家に引っ越し、潮鳴学園に編入する。

**特殊能力**: 残響聴覚（ざんきょうちょうかく）
自らが「沈黙の罪」を背負っているからこそ、他者の飲み込んだ言葉＝残響が「聞こえる」。
この能力により残響界に干渉でき、複数の「残響体（エコーフォーム）」を使役できる。

## 4. ゲームシステム設計

### 4-1. 時間システム

**1日の構造**:

| 時間帯 | 内容 | 行動回数 |
|---|---|---|
| 朝 | 登校イベント・ランダム会話 | 自動進行 |
| 授業 | 授業イベント・クイズ（知性UP） | 自動進行（選択肢あり） |
| 放課後 | 自由行動 | 1回 |
| 夜 | 自由行動 | 1回 |
| 深夜 | 自室のみ（勉強・読書・道具整理） | 1回（制限あり） |

放課後と夜が主要な行動ポイント。1日に最大3行動（放課後・夜・深夜）だが、ダンジョン探索は放課後＋夜を消費する（ペルソナ同様、体力消費で夜は行動不能になる場合あり）。

**時間経過イベントトリガー（例）**:
1. 4/15: 最初の歪み発生 → 残響界への初侵入イベント
2. 5/20-25: 中間テスト期間 → テスト結果で知性ボーナス
3. 7/20: 夏祭りイベント → 共鳴キャラとの特別会話解放
4. 8/15: 海での花火イベント → メインストーリー転換点
5. 10/31: 学園祭 → 全共鳴キャラとの大型イベント
6. 12/24: クリスマスイベント → 特定キャラとの特別シーン
7. 1/1: 初詣イベント → 主人公の過去回想
8. 2/14: バレンタインイベント → 恋愛関係キャラとの分岐
9. 3/1: 卒業式前 → ファイナルダンジョン期限

**月次デッドライン（ボス戦期限）**:
各章で「歪み」が臨界に達する期限が設定される。期限内にダンジョンボスを倒さないと、対象NPCが「言葉を失う」（バッドイベント発生）。

| 章 | 期限 | ダンジョン | テーマ |
|---|---|---|---|
| 第1章 | 5月末 | 沈黙の教壇 | 教師の飲み込んだ怒り |
| 第2章 | 7月中旬 | 仮面の舞台 | 演劇部員の偽りの笑顔 |
| 第3章 | 9月末 | 凍れる波止場 | 漁師の老人の後悔 |
| 第4章 | 11月末 | 反響の画廊 | 画家の表現への恐怖 |
| 第5章 | 1月中旬 | 虚ろの楽譜 | 音楽家の封じた旋律 |
| 最終章 | 3月1日 | 沈黙の書庫 | 全ての残響の根源 |

### 4-2. 戦闘システム

**基本形式**: ターン制コマンドバトル（プレスターン系）

**属性システム**:

| 属性 | 記号 | 色 |
|---|---|---|
| 火炎（カエン） | 🔥 | 赤 |
| 氷結（ヒョウケツ） | ❄️ | 青白 |
| 疾風（シップウ） | 🌀 | 緑 |
| 雷撃（ライゲキ） | ⚡ | 黄 |
| 光輝（コウキ） | ✨ | 白金 |
| 暗黒（アンコク） | 🌑 | 紫黒 |
| 共鳴（キョウメイ） | 🎵 | 藍 |
| 物理 | ⚔️ | 灰 |

「共鳴」はこのゲーム固有の属性。残響体の感情的な波動による攻撃。社会リンクの深さに応じて威力が上がる独自要素。

**弱点ヒット時の効果: "アンコール（Encore）"**
弱点を突くと「アンコール」が発動し、追加行動を1回得る。
（ペルソナのOne More / プレスターンに相当）

連続アンコールでパーティ全員が弱点を突くと「グランドフィナーレ」が発動可能に。

**グランドフィナーレ（Grand Finale）** ※All-out Attack相当:
全員が敵に一斉攻撃するカットイン演出。
- 発動条件: 全敵がダウン状態
- 演出: 画面が一瞬暗転 → 各キャラの攻撃カットインが高速で切り替わる → 最後にパーティ集合の一枚絵 → ダメージ表示
- 一枚絵はパーティ編成によって変化（ペルソナ5のAll-out Attackフィニッシュ絵に相当）

**主人公の特殊能力: "残響体（エコーフォーム）"切り替え**
- 主人公のみ、複数の残響体を所持・切り替え可能（ペルソナのワイルド能力に相当）
- 残響体はダンジョン内で「共鳴」して仲間にするか、「合奏（フュージョン）」で合成する
- 他パーティメンバーは固有の残響体1つのみ（成長で進化する）

**合奏（フュージョン）システム**:
- 2体以上の残響体を組み合わせて新しい残響体を生み出す
- 共鳴ランクが高いアルカナの残響体にボーナス経験値
- 特定の組み合わせで「特殊合奏」（レアな残響体）が出現

## 5. キャラクター設計（パーティメンバー詳細）

### 主人公: 水瀬 蓮（みなせ れん）

**年齢**: 17歳（高2） **外見**: 黒髪を右目にかかるように流す。ヘッドフォンを首にかけている。制服のネクタイは緩め。
**初登場シーン**: 雨の日、潮鳴駅に降り立つ。叔母に出迎えられ、「ここでは静かに暮らしなさい」と言われる。駅を出た瞬間、誰もいないはずのホームから「……助けて」という声が聞こえる。

**表の顔**: 無口で穏やか。争いを避け、誰にも深入りしない。
**真の傷**: 親友が苦しんでいた時、「今さら何を言っても」と沈黙した。その沈黙が親友を追い詰めた。

## 6. ストーリー構成

### 3幕構成

**第1幕: 「沈黙」（4月〜6月）**

蓮が潮鳴市に引っ越し、潮鳴学園に編入する。
他者の残響が聞こえる能力に戸惑う中、スマホに謎のアプリ「Reverb」がインストールされる。
最初の歪みが学園内で発生——国語教師・瀬戸の教室が残響界と繋がり、授業中に生徒が次々と無気力になる。
蓮は悠斗・凛と共に残響界に突入し、瀬戸の飲み込んだ怒り（生徒に言いたくても言えなかった本音）と対峙する。

第1幕の核心: 「残響界とは何か」「なぜ蓮に残響が聞こえるのか」の謎が提示される。
蓮自身は「自分は聞くだけでいい。もう二度と、言葉で誰かを傷つけない」と決意しているが、残響界での戦いがその信念を揺さぶり始める。

**第2幕: 「共鳴」（7月〜11月）**

歪みが潮鳴市全体に広がり始める。
各ダンジョン攻略を通じて紬・壮馬がパーティに加わる。
共鳴（社会リンク）が深まるにつれ、蓮自身の過去が断片的に明かされる。

8月の花火大会で転換点——謎の転校生・御影透子が蓮に告げる。
「あなたの沈黙が、この街の残響界を目覚めさせた」。
蓮の過去の「沈黙の罪」こそが、潮鳴市の残響界を活性化させた触媒だったと判明。

蓮は激しく動揺するが、仲間たちが「お前のせいじゃない。でも、お前にしか終わらせられない」と支える。

**第3幕: 「残響（のこりひびき）」（12月〜3月）**

残響界の深部で真実が明らかになる。
残響界の核には「沈黙の書庫」——人類が飲み込んできた全ての言葉が蓄積された空間がある。
その管理者「ヴォイドキーパー」は、「言葉は人を傷つける。だから沈黙こそが平和だ」と主張し、全人類から「声」を奪おうとしている。

**どんでん返し①**:
ヴォイドキーパーの正体は、蓮の親友・大和の残響体。
大和は蓮が沈黙したことで深く傷つき、その痛みから「もう誰も傷つかないように、全員を黙らせよう」という歪んだ願いが残響界の核を暴走させた。
伏線: 序盤から蓮の夢に出てくる「声のない少年」が大和。蓮のスマホに時々届く差出人不明のメッセージは大和の残響。

**どんでん返し②**:
透子は「前回の残響界の崩壊」を生き延びた存在。
数十年前にも同じ事態が起き、その時は残響界ごと「封印」して解決した。しかし封印は根本的解決ではなく、蓄積された残響がいつか臨界を超える。
透子は「今度こそ完全に残響界を消滅させる」ために蓮に近づいたが、それは蓮の能力を犠牲にすること＝蓮が二度と他者の声を聞けなくなることを意味する。
伏線: 透子が時々「昔もこんなことがあった」とぼやく発言。潮鳴市の歴史資料に「数十年前に住民が一斉に無気力化した事件」の記録。

### エンディング分岐

**ノーマルエンド**: 
ヴォイドキーパーを倒し、残響界の暴走を止める。しかし残響界自体は消えず、蓮は「聞く者」として残り続ける。潮鳴市を去り、新しい場所で静かに暮らす。
条件: 最終ダンジョンをクリアする。

**トゥルーエンド**:
蓮が大和の残響に向かって「あの時、言えなかった言葉」を伝える。
「お前が苦しんでるの、分かってた。でも怖くて言えなかった。——ごめん。そして、ありがとう。お前は俺の親友だ」
この「飲み込んでいた言葉」を声にすることで、残響界の核が浄化される。残響界は消滅するが、人々は再び自分の言葉で語り始める。蓮は潮鳴市で仲間と共に過ごす未来を選ぶ。
条件: 全パーティメンバーの共鳴ランク10 + 全ステータス★★★★以上 + 特定の選択肢で「自分の言葉で語る」を選び続ける。

**バッドエンド**:
蓮がヴォイドキーパーに同意し、「沈黙」を選ぶ。「言葉は人を傷つける。なら、誰も何も言わなければいい」。世界から声が消え、残響界が現実を飲み込む。
条件: 最終選択で「黙る」を選ぶ。

**ペルソナに勝っている点**:
- 「飲み込んだ言葉」というテーマは、ペルソナの「仮面」よりも日常的かつ普遍的。プレイ後に現実で行動を変えたくなるテーマ力。
- 共鳴属性が社会リンクの深さに連動する設計により、システムとナラティブの融合がペルソナ以上に密結合。

**このゲームが最も力を入れるべき点**:
キャラクターの台詞の質。「声に出して読んで感情が動くか」を全台詞で検証する。
台詞の一行一行がプレイヤーの心を撃ち抜かなければ、どれだけシステムが良くても感情移入は生まれない。
