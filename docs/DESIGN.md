## 1. タイトル・コアコンセプト

**ゲームタイトル**: 残響のアリア / Aria of Echoes
**キャッチコピー**: 「聞こえるか――お前が殺した声が。」
**コアテーマ**: 人は「聞かなかったふり」をした声にいつか飲み込まれる。沈黙は共犯である。
**プレイヤーに最終的に感じさせたい感情**: 「自分も誰かの声を無視していたかもしれない」という痛みと、「それでも声を上げ続ける」という静かな決意。

### コアテーマの詳細

ペルソナ5が「反逆」、ペルソナ4が「真実」、ペルソナ3が「死」をテーマにしたように、本作のテーマは **「沈黙と共鳴」** である。

現代社会で人々は日々、誰かの苦しみの声を「聞かなかったふり」をしている。いじめを見て見ぬふり、SNSの炎上に加担する沈黙、助けを求める声を「面倒だから」と無視する日常。その「聞かなかった声」が異世界で怪物となって襲いかかる。

主人公たちは「共鳴者（レゾナント）」として、沈黙に飲み込まれた人々の声を取り戻す。

## 2. 技術スタック

**エンジン**: Unity 6 (6000.1.x LTS)
**言語**: C#
**ターゲットプラットフォーム**: PC (Steam) / Nintendo Switch

### 選定理由
- 2Dと3Dのハイブリッド演出（フィールド3D・戦闘カットイン2D）に最適
- UniTaskによる非同期処理でペルソナレベルの演出チェーンが実装可能
- DOTweenによるUIアニメーションの表現力
- ScriptableObjectによるデータ駆動設計との親和性が高い
- 個人〜少人数開発での生産性

### 必須パッケージ
- UniTask（async/await非同期処理）
- DOTween Pro（アニメーション・演出）
- TextMeshPro（テキスト描画）
- Addressables（アセット管理・シーン管理）
- Cinemachine（カメラ演出）
- Input System（新入力システム）
- Universal Render Pipeline (URP)

## 3. 世界観・舞台設定

### 舞台

**都市名**: 静岡県・響ヶ崎市（ひびきがさきし）
**学校名**: 県立 響ヶ崎総合高等学校
**時代**: 現代（202X年）
**季節**: 4月〜翌3月（1年間）

響ヶ崎市は駿河湾を望む中規模の港町。かつて繁栄した造船業は衰退し、再開発と旧市街の軋轢が街を二分している。海沿いの廃墟ビル群と、駅前の真新しい商業施設の対比が、この街の「聞かれなかった声」を象徴する。

高校は丘の上にあり、通学路から海と旧市街が一望できる。美しい風景の中に、見て見ぬふりをされた痛みが溶け込んでいる。

### 二重世界設定

**異世界名**: 沈黙層（サイレント・ストラタム）

人々が「聞かなかったふり」をした声・感情・叫びが沈殿する階層世界。現実世界の裏側に、音が死んだ空間として存在する。

**異世界が存在する理由**:
人間の集合無意識の中で、「伝えられなかった想い」「無視された叫び」は消滅せず、沈黙層に堆積する。十分な量の沈黙が蓄積すると「残響（エコー）」という怪物が生まれ、現実世界にも影響を及ぼし始める。

**視覚的イメージ**:
- 基調色: 深い藍色と灰白色。音が存在しない世界の「静けさ」を色で表現
- 質感: 水中のような浮遊感。音波の紋様が空間に凍りついている
- 空気感: 完全な無音から始まり、進むほどに歪んだ残響が聞こえ始める
- 各階層は「沈黙の主」の感情に応じて風景が変化する
  - 怒りの沈黙: 赤黒い鋼鉄の迷宮、壁が脈打つ
  - 悲しみの沈黙: 青白い氷の空間、涙のように水滴が浮遊
  - 諦めの沈黙: 灰色の砂漠、すべてが砂に還る

**侵入条件・方法**:
主人公のスマートフォンに現れる謎のアプリ「レゾナンス」。特定の場所で「聞こえない声」が一定以上蓄積した時、アプリが反応し、沈黙層への門が開く。侵入時は周囲の音が段階的に消えていき、完全な無音になった瞬間に異世界へ転移する。

### 主人公

**名前**: 柊 奏汰（ひいらぎ かなた）/ デフォルト名、変更可能
**年齢**: 17歳（高校2年生）
**外見**: 黒髪、左耳にだけヘッドフォンをかけている（右耳は常に開けている）
**特殊能力**: 「共鳴（レゾナンス）」 — 他者の「聞かれなかった声」を聴く力

**能力の理由**:
幼少期に親友が自分に助けを求めていたことに気づけず、その親友が転校（実際は家庭の問題で施設に入った）して以来、「聞こえなかった声」に対する強い罪悪感を抱えている。その罪悪感が沈黙層と共鳴し、他者の沈黙を「聴く」力として覚醒した。

左耳のヘッドフォンは「自分だけの音楽に逃げていた過去の自分」の象徴であり、物語が進むにつれて外す場面が増えていく。

**戦闘での特殊能力**: 複数の「残響体（レゾナンス・フォーム）」を切り替えて戦う。他のキャラクターは1体の残響体しか持てないが、主人公は「すべての声を聴ける」ため、複数の残響体と共鳴できる。

## 4. ゲームシステム設計

### 4-1. 時間システム

**1日の構造**:

| 時間帯 | 行動ポイント | 主な活動 |
|--------|------------|---------|
| 朝（通学） | 0 | イベント・会話のみ。伏線の仕込み |
| 授業中 | 0 | 授業イベント（質問応答でステータスUP） |
| 昼休み | 1 | 短い交流（絆の種まき） |
| 放課後 | 1 | メインの自由行動。絆・ダンジョン・ステータス |
| 夜 | 1 | 家での活動・夜の街・特定キャラとの交流 |
| 深夜 | 0（条件付き1） | 特定条件で沈黙層の深部に潜れる |

**時間経過イベントトリガー**:
1. **月初め**: 新しい「沈黙の主」の噂が学校で広まる（新ダンジョン解放）
2. **中間テスト/期末テスト**: テスト期間は自由行動制限、知識ステータスで結果変動
3. **満月の夜**: 沈黙層が活性化し、強力な残響が出現（ミニボス戦）
4. **季節イベント**: 夏祭り・文化祭・クリスマス等。絆キャラとの特別イベント
5. **デッドライン**: 各章のボス（沈黙の主）には攻略期限がある。期限を過ぎると主が現実に侵食し、バッドイベント発生
6. **天候連動**: 雨の日は沈黙層が不安定になり、レアエネミー出現。晴れの日は特定の場所で釣りやバイトが可能
7. **SNSイベント**: ゲーム内SNS「エコー」で世間の声が流れ、次の事件の伏線になる

**月次デッドライン設計**:
各章（約1〜1.5ヶ月）に1人の「沈黙の主」がいる。期限内にダンジョン攻略＋ボス撃破が必要。期限は月末固定ではなく、ストーリー上の「声が完全に消える日」として設定。期限が近づくと街の雰囲気が変わり、BGMにノイズが混じる演出。

### 4-2. 戦闘システム

**基本形式**: ターン制コマンドバトル（素早さ順行動）

**属性システム「声の属性」**:

| 属性 | 象徴する感情 | 色 |
|------|------------|-----|
| 叫（シャウト） | 怒り・抗議 | 赤 |
| 嘆（ラメント） | 悲しみ・喪失 | 青 |
| 囁（ウィスパー） | 不安・恐怖 | 紫 |
| 諧（ハーモニー） | 喜び・調和 | 緑 |
| 轟（サンダー） | 衝撃・驚愕 | 黄 |
| 寂（サイレンス） | 虚無・諦め | 灰 |
| 響（レゾナンス） | 共鳴・万能 | 白（主人公専用） |

**弱点システム「ワンモア・エコー」**:
弱点を突くと「エコー」（追加ターン）が発生。敵全員をダウンさせると「ユニゾン・バースト」（全体攻撃演出）が発動可能。

- 弱点ヒット → ダウン＋エコー（追加ターン）
- 耐性ヒット → ターン消費が増加（テンポペナルティ）
- 反射 → ダメージが返る＋ターンロスト
- 吸収 → HP回復される＋ターンロスト

**ユニゾン・バースト（All-out Attack相当）**:
敵全員ダウン時に発動可能。パーティ全員の残響体が共鳴し、全体攻撃を放つ。演出はキャラごとの台詞カットイン → 音波が収束 → 一斉解放 → 最後に攻撃したキャラの決めポーズ＋スプラッシュアート。BGMが一瞬無音になってから爆発的に入る演出。

**主人公の特殊能力「多重共鳴」**:
主人公は複数の残響体（他作のペルソナに相当）を所持・切り替え可能。沈黙層で倒した残響を「聴き取る」ことで、その声を自分の力にできる。残響体同士を「調律（チューニング）」で合体させ、新たな残響体を生み出せる（ペルソナ合体に相当）。

**戦闘の感情設計**:
- 雑魚戦: 弱点パズルを解く爽快感。3ターン以内に終わるテンポ
- ボス戦: 沈黙の主の「声」が戦闘中に語られ、感情が揺さぶられる
- ユニゾン・バースト: カタルシスの頂点。毎回気持ちいい演出

### 4-4. ステータス成長システム「五感（ファイブ・センス）」

## 6. ストーリー構成

### 3幕構成

**第1幕: 覚醒（4月〜7月）— 「声を聴く」**

- 4月: 主人公が響ヶ崎に引っ越し（祖父の家に居候）。学校生活開始
- 5月: 最初の「沈黙の主」事件。生徒が突然声を失う現象。アプリ「レゾナンス」の出現。初めて沈黙層に入る。凛花が覚醒

**第2幕: 葛藤（8月〜12月）— 「声を上げる痛み」**

- 8月: 夏休みイベント。海・花火・合宿。仲間との絆深化。しかし街で「声を失う人」が増加
- 9月: 学校内で沈黙が蔓延。声を上げた生徒がSNSで叩かれる事件。テーマ：「声を上げることのリスク」

- 11月: 十六夜の真意が判明。彼は「声を上げても無駄だ」と絶望し、沈黙層の存在に協力していた。主人公チームとの対立と和解
- 12月: 黒幕の存在が浮上。街全体を沈黙で覆おうとする計画。クリスマスイベント（絆キャラとの特別な夜）

## 付録: 設計原則（迷った時に立ち返る）

1. **沈黙は敵であり、テーマであり、演出である** — 音の有無を常に意識する
2. **キャラクターの「聞かれなかった声」が全ての起点** — システムもストーリーもここから生まれる
3. **プレイヤーに「自分も沈黙していたかも」と思わせる** — 説教ではなく、気づきとして
4. **時間は有限。だからこそ、誰の声を聴くかが「選択」になる** — 時間システムとテーマの融合
5. **演出の無音は最大の演出** — ペルソナ5の赤が「うるさい」なら、本作の藍は「静かに深い」
