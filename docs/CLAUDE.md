## 設計メモ（意思決定の記録）

### 2026-06-05 初期設計決定
- **エンジン**: Unity 2022.3 LTS。C#ネイティブ + JRPGの実績 + ScriptableObject駆動が決め手
- **世界観**: 「声」と「残響」を全モチーフの中心に。ペルソナの「仮面/シャドウ」との差別化
- **属性設計**: 感情ベースの8属性。「弱点を突く＝感情の急所を突く」というメタファー
- **捨てた案**: 時間ループ（P3に近すぎる）/ 夢世界（ありきたり）/ SNS二重生活（視覚差別化困難）
- **エコー・リレー追加**: リサーチでP5のBaton Passが戦闘の爽快感に不可欠と判明。エコー・リレーとして導入
- **キャラクター設計方針**: 全員「社会的仮面と本音の乖離」を持つ。ペルソナの「シャドウ対峙」を「ノイズ対峙」として再構築
