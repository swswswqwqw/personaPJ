## 設計メモ（意思決定の記録）

### 2026-06-05: 初期設計完了

**タイトル確定**: 残響のコード / CHORD OF ECHOES

**エンジン選定**: Unity 2022.3 LTS
→ DOTween + UniTask によるキネティックUI、音波可視化ShaderGraphのために選定

**コアコンセプト確定**:
「表現されなかった感情は世界を侵食する──それを声にするとき、人は初めて自由になれる」

**主人公の「声を失いかけている」設定**:
プレイヤーの沈黙と主人公の沈黙がメタ的に一致する設計。全ゲームシステムとナラティブをこの一点に収束させる。

**どんでん返し2件確定**:
① 松永先生の二重スパイ（11月）── 信頼と疑惑の複雑な感情
② 大いなる静寂の正体が主人公自身の感情（2月）── 自己責任と自己受容

**感情周波数属性（6属性）確定**:
喜 / 哀 / 怒 / 恐 / 愛 / 虚 の循環相関。UIでは音波グラフ形式で表示。

**和音（Waon）システム**:
社会リンク相当。8人 + 隠し2人（6月・11月解放）= 計10人。

**UIカラーパレット確定**:
深海青 #0B2240 を軸に、琥珀 #E8A84B をアクセントとして使用。
